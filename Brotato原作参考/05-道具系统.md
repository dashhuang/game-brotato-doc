# Brotato道具系统

## 道具系统概述

道具是Brotato中构筑的核心组成部分，提供各种增益效果，与武器、角色配合产生强大的协同作用。

## 道具分类

### 1. 可叠加道具（Stackable Items）
- **特点**：
  - 可以多次购买同一个道具
  - 效果叠加累积
  - 无购买次数限制
  - 构建极端化构筑的基础

- **使用策略**：
  - 专注堆叠特定属性
  - 创造极端化构筑
  - 适合明确方向的玩法

### 2. 唯一道具/独特道具（Unique Items）
- **特点**：
  - 每种道具只能购买一次
  - 购买后不再出现在商店中
  - 通常效果独特且强力
  - 可能改变构筑方向

- **使用策略**：
  - 根据当前构筑选择合适的唯一道具
  - 某些唯一道具可以开启新的构筑思路
  - 需要理解道具的深层机制

## 道具类型

### 属性强化类道具

提升角色的基础属性，效果直接且稳定。

#### 生存属性
1. **最大生命值提升**
   - 增加生命上限
   - 最基础的生存保障
   - 几乎所有构筑都需要

2. **护甲提升**
   - 减少受到的伤害
   - 按百分比或固定值减伤
   - 后期对抗高伤害敌人的关键

3. **闪避提升**
   - 增加闪避攻击的概率
   - 高闪避可以大幅提升生存能力
   - 但依赖概率，有不稳定性

4. **生命恢复提升**
   - 增加每5秒的生命恢复量
   - 提升持续作战能力
   - 配合高生命值效果更好

5. **生命偷取提升**
   - 攻击时有概率恢复生命
   - 数值为1时=1%概率恢复1点
   - 数值为100时=100%概率恢复1点
   - 高攻速构筑的核心属性

#### 攻击属性
1. **伤害加成（%）**
   - 基础伤害
   - 近战伤害
   - 远程伤害
   - 元素伤害
   - 针对性提升特定类型伤害

2. **攻击速度提升**
   - 增加所有武器的攻击频率
   - 提升DPS的直接方式
   - 配合生命偷取效果显著

3. **暴击率提升**
   - 增加暴击触发概率
   - 配合暴击伤害提升

4. **暴击伤害提升**
   - 增加暴击倍率
   - 需要配合足够的暴击率

#### 特殊属性
1. **幸运值提升**
   - 提高商店出现高品质物品的概率
   - 增加某些特效触发率
   - 影响掉落品质

2. **收获提升**
   - 增加资源掉落量
   - 前期快速积累资源
   - 后期优先级降低

3. **工程提升**
   - 增强召唤物和特殊效果
   - 工程师角色的核心属性
   - 影响多种道具效果

4. **移动速度提升**
   - 提高角色移动速度
   - 改善走位和生存能力
   - 某些构筑的必需品

### 特殊效果类道具

提供独特的游戏机制或效果，是构筑多样化的关键。

#### 已知道具示例

##### 树（Tree）
- **效果**：生成更多树
- **用途**：地形障碍，可能提供掩护或阻挡敌人

##### 惊恐香肠（Scared Sausage）
- **效果**：攻击有25%几率造成1×3燃烧伤害
- **用途**：添加燃烧效果，持续伤害

##### 蛇（Snake）
- **效果**：
  - 燃烧会扩散到附近的一个敌人
  - -1最大生命
- **用途**：
  - 配合燃烧效果，范围伤害
  - 有代价的强力效果

##### 小象（Baby Elephant）
- **效果**：
  - 拾取材料时有+15%几率对随机敌人造成1点伤害（受幸运加成10%）
  - -1基础伤害
- **用途**：
  - 资源收集时附带伤害
  - 需要权衡伤害损失

##### 秘密守门人（Secret Gatekeeper）
- **效果**：
  - 生成一个召唤物
  - 向周围发射5个穿刺闪电弹道
  - 每个弹道造成12点伤害（受工程值影响）
- **用途**：
  - 召唤物提供持续输出
  - 工程构筑的核心道具

## 道具获取

### 获取途径

1. **商店购买**
   - 主要获取方式
   - 使用金币购买
   - 可以刷新商店内容

2. **宝箱奖励**
   - 战斗中可能出现宝箱
   - 开启获得随机道具
   - 通常品质较高

3. **特殊事件（如果有）**
   - 某些特殊情况下的奖励

### 道具品质系统

道具也有品质分级，影响：
- 效果强度
- 购买价格
- 出现概率

## 道具与构筑

### 构筑核心道具

某些唯一道具可以定义整个构筑方向：

1. **燃烧流构筑**
   - 核心道具：蛇、惊恐香肠等
   - 目标：最大化燃烧伤害和扩散
   - 配合：燃烧伤害加成、范围效果

2. **召唤流构筑**
   - 核心道具：秘密守门人等召唤类道具
   - 目标：大量召唤物提供持续输出
   - 配合：工程属性、召唤物增强道具

3. **生命偷取流**
   - 核心道具：生命偷取提升
   - 目标：通过高攻速和偷取实现不死
   - 配合：攻速提升、多武器

4. **暴击流构筑**
   - 核心道具：暴击率和暴击伤害提升
   - 目标：通过高额暴击秒杀敌人
   - 配合：高伤害武器、幸运属性

5. **坦克流构筑**
   - 核心道具：生命、护甲、生命恢复
   - 目标：极致的生存能力
   - 配合：持续输出型武器

### 道具选择策略

#### 前期（1-5波）
- 优先生存属性（生命值、护甲）
- 适当提升收获，积累资源
- 建立基础输出能力

#### 中期（6-10波）
- 明确构筑方向
- 选择核心唯一道具
- 平衡输出和生存

#### 后期（11-15波）
- 专精化构筑
- 堆叠关键属性
- 减少收获投资，专注战斗属性

#### 终局（16-20波）
- 极限强化核心机制
- 补足构筑短板
- 应对最强敌人

## 道具协同机制

### 道具+武器协同

- **燃烧道具 + 范围武器**：快速扩散燃烧
- **攻速道具 + 多武器**：极限DPS
- **工程道具 + 召唤武器**：召唤大军

### 道具+角色协同

- **幸运道具 + 幸运角色**：极高品质掉落
- **工程道具 + 工程师**：召唤流核心
- **暴击道具 + 高暴击角色**：爆发伤害

### 道具+道具协同

- **燃烧扩散 + 燃烧伤害提升**：范围持续伤害
- **生命偷取 + 攻速提升**：持续恢复
- **收获提升 + 幸运提升**：资源滚雪球

## 道具平衡设计

### 平衡原则

1. **代价机制**
   - 强力道具通常有负面效果
   - 需要玩家权衡利弊
   - 例：蛇提供燃烧扩散，但-1生命

2. **递减效应**
   - 同一属性堆叠过多，边际收益递减
   - 鼓励多样化构筑
   - 避免单一属性无脑堆叠

3. **唯一性限制**
   - 强力道具设为唯一
   - 防止无限叠加
   - 保持游戏平衡

4. **价格调节**
   - 强力道具价格更高
   - 需要合理分配资源
   - 不能无脑购买所有好道具

## 道具设计建议

基于Brotato的道具系统，设计道具时应考虑：

1. **功能性**：道具应有明确的功能和适用场景
2. **协同性**：鼓励与其他道具、武器、角色的协同
3. **代价性**：强力效果应伴随一定代价
4. **趣味性**：独特的机制比纯数值提升更有趣
5. **平衡性**：避免必选道具和废弃道具的存在
