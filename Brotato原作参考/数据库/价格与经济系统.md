## 🔑 核心价格公式（Wiki确认）✅

### 商店物品价格公式

```
最终价格 = (基础价格 + 波次 + (基础价格 × 0.1 × 波次)) × 商店价格修正
```

**说明**：
- **基础价格（Base Price）**：每个道具/武器的固定基础价格
- **波次（Wave）**：当前波次数（1-20）
- **通胀系数**：0.1（每波增加基础价格的10%）
- **商店价格修正（Shop Price）**：受Coupon道具和某些角色影响

**最低价格**：1材料（向下取整后最低为1）

### 示例计算

**假设**：酸（Acid）基础价格65

| 波次 | 计算 | 价格 |
|------|------|------|
| 波次1 | (65 + 1 + 65×0.1×1) × 1 = 72.5 | **72材料** |
| 波次5 | (65 + 5 + 65×0.1×5) × 1 = 102.5 | **102材料** |
| 波次10 | (65 + 10 + 65×0.1×10) × 1 = 140 | **140材料** |
| 波次20 | (65 + 20 + 65×0.1×20) × 1 = 215 | **215材料** |

**结论**：后期道具价格显著上涨（约3倍）

### 刷新价格公式

```
刷新递增值 = 向下取整(波次 × 0.40)，最小值为1
首次刷新价格 = 向下取整(波次 × 0.75) + 刷新递增值
每次刷新增加 = 刷新递增值
```

## 🛒 商店机制详解

### 锁定机制（Lock）

**功能**：
- 免费锁定道具/武器
- 锁定后刷新时保留
- 锁定后跨波次保留
- **锁定的物品价格不受通胀影响**（重要！）

**策略价值**：
- 在低波次锁定想要的物品
- 避免后期高昂价格
- 跨波次保留选择

### 回收机制（Recycle）

**回收价值**：
- 基础回收：**25%返还**
- Recycling Machine道具：+35%（总计60%）
- Entrepreneur角色：再+25%（总计85%）
- 基于当前价格（包括通胀）
- 受商店价格修正影响

**公式**：
```
回收获得 = 当前价格 × 回收比例（25%/60%/85%）
```

**注意**：无限模式下，商店价格会大幅上涨，但回收价值不会随之增加。

### 免费刷新（Dangerous Bunny道具）

**危险兔子（Dangerous Bunny）道具**：
- 每个提供1次免费刷新
- 可叠加
- 节省材料
