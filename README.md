## 🎮 游戏简介

《星河舰长》是一款结合了 **Brotato核心玩法** 与 **二次元太空战舰** 题材的游戏。玩家将扮演舰队指挥官，率领各型战舰和英勇的女性舰长们，在星河中抵御四大势力的入侵。

### ✨ 核心特色

| 特色 | 说明 |
|------|------|
| 🎯 **自动射击** | 专注走位，战舰自动锁定攻击 |
| 🚢 **48艘战舰** | 每艘战舰对应一位可养成的女性舰长 |
| ⚔️ **68种武器** | 近程炮/导弹/激光/舰载机四大类 |
| 🔧 **144种模块** | 丰富的Build构建可能性 |
| 👾 **四大势力** | 虫族、机械叛军、星际海盗、异星帝国 |
| 💕 **舰长养成** | 立绘、语音、好感度、专属剧情 |

```
Phase 1: 核心原型
├── 战舰移动 & 自动射击
├── 敌舰AI & 战役生成

└── 目标：可玩Demo

├── 全部武器 & 模块
├── 全部战舰 & 敌人
├── 难度系统
└── 目标：完整玩法循环

Phase 3: 养成系统
├── 舰长立绘 & 语音
├── 好感度 & 剧情

└── 目标：完整游戏体验

Phase 4: 优化上线
├── 特效 & 音效
```
