# 星河舰长 (Galaxy Captain) 游戏策划案

> 一款二次元太空战舰题材的俯视角自动射击Roguelike游戏

## 🎮 游戏概述

《星河舰长》是一款结合了Brotato核心玩法与二次元太空战舰题材的游戏。玩家将扮演舰队指挥官，率领各型战舰和英勇的女性舰长们，在星河中抵御四大势力的入侵。

### 核心特色

- **自动射击战斗**：专注于战术走位，战舰自动锁定攻击
- **48艘战舰 × 48位舰长**：每艘战舰对应一位可养成的女性舰长
- **67种舰载武器**：近程炮/导弹/激光/舰载机四大类型
- **135种舰船模块**：丰富的Build构建可能性
- **四大敌对势力**：虫族、机械叛军、星际海盗、异星帝国
- **舰长养成系统**：立绘、语音、好感度、剧情

## 🎨 美术风格

- **主题**：帅气科幻、机械美学
- **参考**：明日方舟、机战系列、高达
- **色调**：冷色调为主（深空蓝、星舰银、能量青）
- **战舰**：流线型、机械感、高辨识度
- **舰长**：英气飒爽、军装风格
