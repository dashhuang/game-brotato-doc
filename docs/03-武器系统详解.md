# Brotato 武器系统详解

> 本文档详细记录《Brotato》中所有武器的完整信息，包括数值、特效、升级机制等。

---

## 一、武器系统概述

### 1.1 武器数量
- **近战武器**：22种
- **远程武器**：25种
- **元素武器**：12种
- **工程武器**：8种
- **总计**：67种武器

### 1.2 武器槽位
- 玩家最多可同时装备**6把武器**
- 所有武器同时攻击（自动瞄准最近敌人）
- 相同武器可以装备多把

### 1.3 武器等级（Tier）

| Tier | 名称 | 颜色 | 属性倍率 | 价格倍率 |
|------|------|------|----------|----------|
| 1 | 普通 | 白色 | ×1.0 | ×1.0 |
| 2 | 优秀 | 绿色 | ×1.25 | ×1.5 |
| 3 | 稀有 | 蓝色 | ×1.5 | ×2.0 |
| 4 | 史诗 | 紫色 | ×1.75 | ×2.5 |

### 1.4 武器升级机制
- 在商店中购买**已拥有的相同武器**会升级
- 升级将武器提升1个Tier
- Tier 4为最高等级（不再升级）
- 升级后基础属性按倍率提升

## 六、武器属性详解

### 6.1 伤害类型

| 类型 | 加成来源 | 武器示例 |
|------|----------|----------|
| **近战伤害** | 近战伤害% | 剑、斧头、拳头 |
| **远程伤害** | 远程伤害% | 手枪、步枪、弓箭 |
| **元素伤害** | 元素伤害% | 喷火器、冰杖、雷光刀 |
| **工程伤害** | 工程学% | 炮塔、无人机、地雷 |

### 6.2 伤害计算公式

```
武器最终伤害 = 基础伤害 
             × Tier倍率 
             × (1 + 伤害%) 
             × (1 + 类型伤害%)
             × (1 + 暴击伤害%) [如果暴击]
```

### 6.3 DPS计算

```
武器DPS = 最终伤害 × 子弹数 / 冷却时间 × (1 + 攻速%)
        × (1 + 暴击率 × (暴击伤害 - 1))
```

## 八、武器获取方式

### 8.1 商店购买
- 每波结束后商店刷新4个物品
- 可花费材料刷新商店
- 物品品质受幸运属性影响

### 8.2 初始武器
- 大部分角色有1把初始武器
- 部分角色有2把或无初始武器
- 初始武器为Tier 1

### 8.3 箱子掉落
- 战斗中随机出现箱子
- 箱子可能掉落武器
- 掉落品质随机
