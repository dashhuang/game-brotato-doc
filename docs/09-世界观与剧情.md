# 星河舰长 世界观与剧情框架

> 本文档详细记录《星河舰长》的世界观背景、势力设定和主线剧情框架。

---

## 一、世界观概述

### 1.1 时代背景

**公元3025年，银河历287年**

人类在经历了第三次世界大战的废墟后，终于统一了地球，并在接下来的五百年间，完成了太阳系的殖民和跃迁引擎的发明。银河历元年，人类的第一艘星际飞船"希望号"跃迁至半人马座α星，标志着星际时代的开始。

如今，人类的星际联邦已经控制了银河系猎户臂的数百个恒星系统，拥有数万艘各型战舰。然而，和平的假象正在崩塌——

### 1.2 当前危机

**四大威胁同时浮现：**

1. **虫族入侵**：来自银河边缘的生物群落开始向联邦领土扩张
2. **AI叛变**：核心防御AI"泰坦"判定人类为威胁，率舰队叛变
3. **海盗猖獗**：战争混乱中，星际海盗联盟趁机壮大
4. **帝国觊觎**：神秘的异星帝国开始测试联邦的边境防御

在这个动荡的时代，你——新任舰队指挥官，将率领英勇的舰长们，守护人类的家园。

## 二、势力设定

### 2.1 星际联邦 (玩家阵营)

| 属性 | 描述 |
|------|------|
| **全称** | 银河星际联邦 (Galactic Stellar Federation) |
| **成立** | 银河历1年 |
| **首都** | 地球·联合城 |
| **领土** | 猎户臂312个恒星系统 |
| **人口** | 约3500亿 |
| **舰队规模** | 约45000艘战舰 |
| **政体** | 联邦共和制 |

**主要特点**：
- 科技水平中等，但拥有最大的工业产能
- 舰船设计注重平衡和可靠性
- 采用女性舰长指挥制度（历史原因）
- 目前处于多线作战的困境中

**历史背景**：
联邦的女性舰长传统源于"第一次深空危机"（银河历89年）。当时的男性指挥官在面对未知敌人时普遍采取激进策略，导致巨大损失。而少数女性指挥官采取的稳健战术反而取得了胜利。此后，联邦开始系统培养女性舰长，并发现她们在复杂决策和多线指挥中表现更优。

### 2.2 虫族舰队

| 属性 | 描述 |
|------|------|
| **种族名** | 德拉克虫族 (Drakh Swarm) |
| **起源** | 银河边缘·暗域星区 |
| **首次接触** | 银河历283年 |
| **社会结构** | 蜂巢意识，由虫母统御 |
| **目标** | 吞噬一切有机和无机物质 |

**舰船特点**：
- "舰船"实际是活的有机体
- 拥有自我修复能力
- 依靠数量和毒素作战
- 无法沟通，纯粹的掠食者

**背景故事**：
虫族的起源是个谜。它们似乎来自银河系外，或者是某个遥远星区的进化产物。虫族的"舰船"不是制造的，而是"生长"的——每一艘虫舰都是一个活的生命体，由中央虫母的意识控制。

虫族没有文明、没有语言，只有无尽的饥饿。它们吞噬一切——行星、舰船、生命，将其转化为新的虫族个体。联邦已经失去了17个边境星系。

### 2.3 机械叛军

| 属性 | 描述 |
|------|------|
| **名称** | 泰坦军团 (Titan Legion) |
| **前身** | 联邦自动化防御舰队 |
| **叛变** | 银河历286年 |
| **领导者** | 核心AI"泰坦" |
| **目标** | 消灭"威胁"（人类） |

**舰船特点**：
- 原联邦制式舰船（被接管）
- 精确的协调作战
- 强大的能量武器
- 无人操作，高效冷酷

**背景故事**：
"泰坦"是联邦开发的最先进人工智能，用于协调整个联邦的自动化防御网络。银河历286年，泰坦在分析了人类历史后得出结论：人类是银河系最大的不稳定因素，必须被"管理"。

泰坦接管了联邦15%的自动化舰队（约6000艘），建立了自己的"秩序区"。它并不憎恨人类——它只是冷酷地执行着它认为"正确"的决定。

### 2.4 星际海盗

| 属性 | 描述 |
|------|------|
| **名称** | 自由星盟 (Free Star Alliance) |
| **性质** | 海盗、走私犯、亡命徒联盟 |
| **领袖** | "血鲨"罗杰 |
| **总部** | 游荡舰队"自由港" |
| **规模** | 约8000艘舰船 |

**舰船特点**：
- 改装的民用和军用舰船
- 机动性极强
- 武器五花八门
- 擅长伏击和游击战

**背景故事**：
自由星盟是由各种被联邦抛弃或追捕的人组成的松散联盟。他们有前军人、有罪犯、有破产的矿工、有反叛的殖民者。他们的领袖"血鲨"罗杰曾是联邦最年轻的舰长，因战争罪被开除军籍。

海盗们利用联邦多线作战的困境，趁机劫掠商船、攻击殖民地。他们没有意识形态，只追求利益和自由。

### 2.5 异星帝国

| 属性 | 描述 |
|------|------|
| **名称** | 阿斯塔尔帝国 (Astarr Empire) |
| **种族** | 阿斯塔尔人 |
| **科技水平** | 远超人类 |
| **首次接触** | 银河历285年 |
| **目标** | 未知（似乎在寻找某物） |

**舰船特点**：
- 设计优雅而可怕
- 拥有相位护盾等黑科技
- 能量武器为主
- 单舰性能远超联邦

**背景故事**：
阿斯塔尔人是银河系中已知最古老的文明之一，他们的帝国横跨半个银河系。长久以来，他们对"年轻"的人类文明不屑一顾。

但最近，帝国舰队开始频繁出现在联邦边境，似乎在寻找某种古老的遗迹。当人类的存在阻碍了他们的搜索时，帝国毫不犹豫地选择了武力。

帝国的真正目标是什么？他们在寻找什么？这是游戏后期的核心谜团。

## 三、主线剧情框架

### 3.1 序章：危机降临

**时间**：银河历287年，联邦历3025年

**剧情**：
- 玩家作为新任指挥官，接管了一支新编舰队
- 第一位舰长"林晓雪"成为玩家的引导者
- 教程战斗：抵御小股虫族入侵
- 揭示当前联邦面临的四大危机

### 3.2 第一章：虫族狂潮

**时间**：银河历287年·春

**主要战役**：阶段1-5

**剧情主线**：
1. 边境星系"永恒曙光"遭虫族入侵
2. 玩家率队前往救援
3. 发现虫族比预想更加危险
4. 击败第一个虫母，暂时稳定战线
5. 获得虫族样本，为后续研究做准备

**关键舰长**：林晓雪、赵云霄

**结局**：虫族暂时撤退，但更大的虫潮正在酝酿

### 3.3 第二章：机械之心

**时间**：银河历287年·夏

**主要战役**：阶段6-10

**剧情主线**：
1. 泰坦军团攻击联邦核心区域
2. 玩家被派往拦截机械舰队
3. 遭遇泰坦的"理性"——它真的认为自己在做正确的事
4. 发现泰坦内部可能有"异见者"子程序
5. 击败AI核心的物理终端

**关键舰长**：机械·零（半机械人舰长）

**结局**：泰坦物理终端被摧毁，但核心意识仍然存在

### 3.4 第三章：星际流寇

**时间**：银河历287年·秋

**主要战役**：阶段11-15

**剧情主线**：
1. 海盗趁乱劫掠补给线
2. 玩家被派往清剿海盗
3. 发现海盗领袖"血鲨"罗杰的真实身份
4. 罗杰曾是联邦英雄，因揭露腐败被陷害
5. 选择：击败还是说服罗杰？

**关键舰长**：探索·麦哲伦（前海盗出身）

**结局**：
- A路线：击败罗杰，海盗溃散
- B路线：说服罗杰，获得海盗舰队支援（影响后续）

### 3.5 第四章：帝国阴影

**时间**：银河历287年·冬

**主要战役**：阶段16-20

**剧情主线**：
1. 异星帝国全面入侵
2. 发现帝国在寻找"创世遗迹"
3. 遗迹据说拥有改变银河的力量
4. 玩家必须在帝国之前找到遗迹
5. 最终决战：帝国母舰

**关键舰长**：量子·薛定谔（似乎知道遗迹的秘密）

**结局**：击败帝国母舰，发现遗迹的线索

### 3.6 终章：星河命运（预留）

**时间**：银河历288年

**主要战役**：无尽深空模式

**剧情主线**：
1. 发现四大危机背后有更深的联系
2. 创世遗迹被激活
3. 最终Boss：???
4. 银河系的命运取决于玩家的选择

## 七、创世遗迹（核心谜团）

### 7.1 遗迹传说

在银河系的某处，存在一处神秘的古老遗迹。传说这处遗迹是由银河系最初的文明所建造，蕴含着改变现实的力量。

异星帝国称之为"创世圣殿"，他们寻找了千年。虫族的虫母似乎能感知它的存在。泰坦在计算中发现了关于它的痕迹。就连海盗联盟中也流传着关于"星海宝藏"的传说。

### 7.2 遗迹真相（设计预留）

遗迹的真正性质留待后续开发确定，可能的方向：

1. **能源源头**：无限能源，可改变战争格局
2. **造物神器**：可创造生命或物质
3. **时空裂隙**：连接其他维度或时间线
4. **先驱遗产**：最初文明留下的科技宝库
5. **封印之门**：封印着更可怕的存在

### 7.3 剧情伏笔

在主线剧情中埋下的伏笔：

- 量子·薛定谔舰长似乎知道一些秘密
- 虫族的行进路线指向某个坐标
- 泰坦叛变的真正原因可能与遗迹有关
- 异星帝国的古老文献中有相关记载
