## 2. 游戏背景

公元3025年，人类的星际联邦遭遇了前所未有的危机——虫族入侵、AI叛乱、海盗猖獗、异星帝国觊觎。在这个动荡的时代，你将作为新任舰队指挥官，率领各型战舰和英勇的女性舰长们，在星河中书写传奇。

---

## 3. 核心特色

### 3.1 自动射击战斗
- 战舰自动锁定攻击最近敌人
- 玩家专注于**战术走位**和**规避伤害**
- 横版视角，便于移动端操作

### 3.2 丰富的Build构建
- **48艘战舰**：各有独特属性和被动技能
- **68种武器**：近程炮/导弹/激光/舰载机四大类
- **144种模块**：火控/防御/引擎/战术等多类型
- **组合可能性**：海量Build组合

### 3.3 Roguelike循环
- 随机掉落、随机补给
- 舰桥升级选择
- 每局体验不同
- 永久解锁进度

### 3.4 舰长养成
- 48位女性舰长对应48艘战舰
- 立绘、语音、好感度、专属剧情
- 誓约系统（好感度满级）

## 7. 易于上手，难于精通 ⭐

这是游戏设计的黄金法则，也是《星河舰长》的核心设计理念。

### 7.1 易于上手

| 设计 | 说明 |
|------|------|
| **自动射击** | 武器自动锁定攻击最近敌人，玩家无需瞄准 |
| **简单操作** | 仅需控制移动（方向键/摇杆），单手可玩 |
| **核心机制清晰** | 打怪→拾取晶体→购买强化→继续打怪 |
| **新手保护** | 危险0难度提供低压力入门体验 |
| **引导战舰** | 先锋号无负面属性，适合新手学习 |

### 7.2 难于精通

| 设计 | 说明 |
|------|------|
| **数值系统深度** | 属性公式、边际效应、阈值设计需要理解 |
| **构筑策略** | 战舰×武器×模块的协同组合需要规划 |
| **资源管理** | 刷新/购买/锁定的经济决策 |
| **走位技术** | 高难度下需要精确走位拾取和规避 |
| **48艘战舰挑战** | 解锁条件和特殊玩法提供长期目标 |

### 7.3 新手友好战舰

以下战舰适合新手入门：

| 战舰 | 原因 |
|------|------|
| **SSA-01 先锋号** ⭐ | 默认解锁，无负面属性，全面均衡 |
| **SSA-03 猎鹰号** | 纯正面加成，无特殊限制 |
| **SSB-06 重巡号** | 火力与防御平衡，容错率高 |
| **SSC-01 堡垒号** | 超高护盾，适合学习走位 |

## 8. 自然教学设计

### 8.1 设计哲学

> "Show, don't tell" - 通过游戏本身教学，不依赖冗长教程

### 8.2 教学机制

| 阶段 | 自然教学内容 |
|------|-------------|
| **阶段1** | 基础操作：移动、自动攻击、拾取晶体 |
| **阶段2** | 舰桥升级：选择属性强化 |
| **阶段3** | 补给站：购买武器和模块 |
| **阶段5** | 首次Boss战：学习优先目标 |
| **阶段6+** | 敌舰类型变化：学习应对不同威胁 |

### 8.3 失败容错

| 机制 | 说明 |
|------|------|
| **掉落保留** | 未拾取的晶体下阶段重新掉落 |
| **危险0难度** | 敌舰属性减半，允许犯错 |
| **紧急护盾模块** | 护盾归0时回复护盾，提供翻盘机会 |
