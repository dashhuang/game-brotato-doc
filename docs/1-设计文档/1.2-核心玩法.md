# 1.2 核心玩法

> 游戏核心循环与战斗机制详解

---

## 1. 单局游戏流程

```
┌─────────────────────────────────────────────────────────────┐
│                         母港                                 │
│  选择战舰 → 选择初始武器 → 确认出击                          │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                      战役阶段 1-20                           │
│  ┌─────────────┐    ┌─────────────┐    ┌─────────────┐      │
│  │  战斗阶段   │ → │  舰桥升级   │ → │   补给站    │       │
│  │  20-90秒   │    │  (满足条件) │    │  (每阶段后) │       │
│  └─────────────┘    └─────────────┘    └─────────────┘      │
│         ↑                                      ↓             │
│         └──────────── 下一阶段 ←──────────────┘             │
└─────────────────────────────────────────────────────────────┘
                              ↓
┌─────────────────────────────────────────────────────────────┐
│                        战斗结算                              │
│  统计数据 → 好感度增加 → 解锁检查 → 返回母港                 │
└─────────────────────────────────────────────────────────────┘
```

## 2. 战斗阶段

### 2.1 基本规则

| 项目 | 说明 |
|------|------|
| **持续时间** | 第1阶段20秒 → 第20阶段90秒（逐步增加） |
| **敌舰刷新** | 从屏幕边缘持续跃迁进入 |
| **视角** | 横版视角，战舰左右移动为主 |
| **自动攻击** | 武器自动锁定最近敌舰 |
| **玩家操作** | 移动、规避、拾取掉落物 |

### 2.2 战斗操作

```
┌─────────────────────────────────────────┐
│           横版战场示意图                 │
│                                         │
│   敌舰 →  ●  ●  ●                      │
│                    ▲                    │
│                   战舰                  │
│                    ↕ 移动               │
│         ●  ●  ● ← 敌舰                 │
│                                         │
│   [方向键/摇杆] 移动战舰                 │
│   武器自动开火                          │
└─────────────────────────────────────────┘
```

### 2.3 掉落物

| 掉落物 | 图标 | 效果 |
|--------|------|------|
| **能量晶体** | 💎 绿色 | 经验值 + 货币（双重功能） |
| **修复包** | ❤️ 红色 | 恢复护盾值 |
| **补给箱** | 📦 金色 | 额外奖励（晶体/模块） |

## 3. 舰桥升级系统

### 3.1 触发条件

- 累计收集的能量晶体达到**升级阈值**时触发
- **注意**：计算的是"累计收集量"，购买物品不影响升级进度

### 3.2 升级选项

每次升级从**4个随机属性**中选择1个强化：

| 属性类型 | 提升内容 | 设计说明 |
|----------|----------|----------|
| 舰体容量 | +5 最大护盾 | 生存核心 |
| 护盾再生 | +1 每5秒回复 | 持续生存 |
| 能量吸收 | **+3%** 攻击回盾概率 | 对标Brotato生命偷取，可达阈值 |
| 火力输出 | +5% 伤害加成 | 通用DPS |
| 射击频率 | +5% 攻速 | DPS核心 |
| 精确打击 | +3% 暴击率 | 爆发输出 |
| 装甲强化 | +1 装甲值 | 稳定减伤 |
| 规避系统 | +3% 规避率 | 概率生存 |
| 战场运势 | +8% 幸运值 | 经济/品质 |
| 资源回收 | **+8%** 晶体掉落 | 前期投资价值高 |
| 火力范围 | +15 攻击范围 | 安全距离 |
| 引擎推力 | +5% 移动速度 | 走位能力 |
| 近程强化 | +10% 近程炮加成 | 专精流派 |
| 导弹强化 | +10% 导弹加成 | 专精流派 |
| 能量武器 | +10% 激光加成 | 专精流派 |
| 舰载指挥 | +10% 舰载机加成 | 专精流派 |

> **阈值设计说明**：能量吸收+3%/次，搭配模块可在约15-20次升级后达到50-60%有效吸收，对标Brotato的"生命偷取点满100"策略。资源回收+8%/次，强调前期投资价值。

## 4. 补给站系统

### 4.1 基本功能

- 每个战役阶段结束后开放
- 显示**4个补给物品**（武器或模块）
- 使用能量晶体购买

### 4.2 补给站操作

| 操作 | 说明 | 费用 |
|------|------|------|
| **装载补给** | 购买武器/模块 | 物品标价 |
| **刷新补给** | 更换4个新物品 | Wiki公式：floor(阶段×0.75)+阶段×0.40，每次+阶段×0.40 |
| **舰载卸载** | 出售已装备物品 | 获得60%价格（Wiki确认） |
| **武器合成** | 购买已拥有的相同武器，2把自动合成为1把更高Tier | 物品标价 |
| **锁定物品** | 防止被刷新替换 | 免费 |

> **刷新成本设计**：采用×2指数递增（对标Brotato），每阶段重置。这是核心经济约束，强迫玩家在有限刷新次数内做出决策，避免无脑刷新直到最优解。
> 
> **武器合成机制（对标Brotato）**：购买第2把相同武器时，2把自动合成为1把更高Tier武器。例如：持有1把T1速射力场，再购买1把T1速射力场 → 自动合成为1把T2速射力场。注意：合成后武器数量减少，需权衡"2把低级"vs"1把高级"的策略价值。

### 4.3 物品品质概率

| 品质 | 基础概率 | 战场运势影响 |
|------|----------|--------------|
| 标准型（白） | 55% | -运势×0.3% |
| 改良型（绿） | 30% | +运势×0.15% |
| 精英型（蓝） | 12% | +运势×0.1% |
| 旗舰型（紫） | 3% | +运势×0.05% |

## 5. 武器合成系统

### 5.1 合成规则

```
同款武器 T1 × 3 → 武器 T2 × 1
同款武器 T2 × 3 → 武器 T3 × 1
同款武器 T3 × 3 → 武器 T4 × 1
```

### 5.2 等级提升

| Tier | 名称 | 属性倍率 |
|------|------|----------|
| T1 | 标准型 | ×1.0 |
| T2 | 改良型 | ×1.25 |
| T3 | 精英型 | ×1.5 |
| T4 | 旗舰型 | ×1.75 |

## 6. 战役阶段设计

### 6.1 时间表

| 阶段 | 持续时间 | 敌人密度 | 势力 | 特殊事件 |
|------|----------|----------|------|----------|
| 1-5 | 20-40秒 | 低→中 | 虫族 | 阶段5：虫母Boss |
| 6-10 | 45-65秒 | 中→高 | 机械叛军 | 阶段10：AI核心Boss |
| 11-15 | 65-75秒 | 高 | 星际海盗 | 阶段15：海盗王Boss |
| 16-20 | 80-90秒 | 极高 | 异星帝国 | 阶段20：帝国母舰Boss |

### 6.2 胜利条件

- **存活至阶段结束**：进入下一阶段
- **通关第20阶段**：游戏胜利
- **战舰损毁**：游戏结束

## 7. 难度系统

### 7.1 危险等级

| 等级 | 敌舰护盾 | 敌舰伤害 | 敌舰数量 | 解锁条件 |
|------|----------|----------|----------|----------|
| 0 | ×0.5 | ×0.5 | ×0.7 | 默认 |
| 1 | ×1.0 | ×1.0 | ×1.0 | 通关1次 |
| 2 | ×1.25 | ×1.15 | ×1.15 | 通关危险1 |
| 3 | ×1.5 | ×1.3 | ×1.3 | 通关危险2 |
| 4 | ×1.8 | ×1.5 | ×1.5 | 通关危险3 |
| 5 | ×2.2 | ×1.75 | ×1.75 | 通关危险4 |

### 7.2 无尽深空模式

- 通关第20阶段后解锁
- 战役阶段无限继续
- 难度持续增加
- 用于刷分和挑战
