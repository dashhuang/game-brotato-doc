# 1.0 设计理念

> 《星河舰长》的核心公式、设计哲学与成功要素

---

## 1. 核心公式

```
星河舰长 = 横版自动射击 + Roguelike + 多武器系统 + 舰长养成 + 二次元美学
```

这是一句话描述游戏本质的核心公式，每个元素缺一不可。

---

## 2. 核心循环

```
战斗（20-90秒）→ 舰桥升级（选择属性）→ 补给站（购买/刷新）→ 下一阶段 → ... → 第20阶段胜利
```

**设计优势**：
- 节奏张弛有度：战斗紧张，补给放松
- 循环简洁易懂
- 单局时长可控（20-30分钟）
- 每2-3分钟就有一次决策点（升级/购买）

---

## 3. 六大成功要素

### 3.1 易于上手，难于精通 ⭐

这是游戏设计的黄金法则。

| 易于上手 | 难于精通 |
|---------|---------|
| 自动射击，无需瞄准 | 数值系统深度 |
| 移动操作简单（仅方向键/摇杆） | 构筑策略多样 |
| 核心机制易懂（打怪、拾取、购买） | 资源管理需要规划 |
| 有新手友好战舰（先锋号、猎鹰号） | 走位技术有上限 |
| 危险0难度提供新手保护 | 48艘战舰提供无限挑战 |

**设计哲学**：
> "分钟学会，一生精通"
> 降低入门门槛，保持高技术上限

### 3.2 高重玩价值

#### 内容多样性
- 48艘战舰，每艘玩法不同
- 68种武器
- 144种模块
- 无数种构筑组合

#### 随机性
- 补给站物品随机
- 敌舰组合随机
- 掉落随机
- 每局体验不同

#### 解锁系统
- 战舰需要解锁
- 提供长期目标
- 持续的进度感

### 3.3 精妙的数值设计

#### 平衡的难度曲线
- 敌人与玩家同步成长（约10倍成长曲线）
- 保持挑战性但不绝望
- 20阶段结构经过精心设计

#### 多维度属性
- 不是单一"战力"数值
- 多种属性提供策略深度
- 鼓励差异化构筑

#### 边际效应
- 同一属性堆叠收益递减
- 避免单一属性无脑堆叠
- 鼓励平衡发展

#### 代价机制
- 强力效果伴随负面属性
- 增加选择深度
- 没有绝对必选项

### 3.4 策略深度

#### 构筑策略
- 战舰选择（定义玩法方向）
- 武器组合（最多6把协同）
- 模块搭配（被动效果叠加）
- 属性分配（舰桥升级选择）

#### 资源管理
- 晶体分配（购买/刷新/锁定等决策）
- 资源回收的阶段性投资
- 短期vs长期收益权衡

#### 战斗策略
- 目标优先级判断
- 走位技巧
- 风险-收益决策（拾取掉落）

#### 适应性
- 根据补给站内容调整构筑
- 根据敌舰类型调整策略
- 灵活应变

### 3.5 反馈清晰

#### 视觉反馈
- 伤害数字（普通白色，暴击放大+特效）
- 击中火花、命中闪烁
- 敌舰爆炸、碎片飞溅
- 掉落物清晰可见

#### 数值反馈
- 属性面板清晰
- 实时战斗数据可查
- 补给站显示属性变化预览

#### 进度反馈
- 阶段进度显示
- 经验条
- 晶体数量
- 护盾状态

### 3.6 控制与混乱的平衡

| 玩家可控 | 随机元素 |
|---------|---------|
| 战舰选择 | 补给站物品 |
| 构筑方向 | 敌舰组合 |
| 资源分配 | 掉落品质 |
| 战斗走位 | 舰桥升级选项 |

**平衡点**：
- 有足够的控制感（不是纯运气）
- 有足够的随机性（保持新鲜感）
- 技术可以弥补运气
- 每局都是独特体验

## 4. 五大设计哲学

### 4.1 简洁而深刻

**表面**：
- 操作简单（移动+自动射击）
- 核心循环清晰
- 规则易懂

**深层**：
- 数值系统复杂
- 策略深度大
- 构筑组合无穷

### 4.2 选择与代价

**所有选择都有代价**：
- 购买A就买不起B（经济代价）
- 堆叠属性X就无法提升Y（属性代价）
- 强力模块伴随负面效果（效果代价）
- 刷新补给站消耗晶体（机会代价）

**哲学**：
- 没有完美的选择
- 增加决策深度
- 鼓励权衡和规划

### 4.3 平衡而非绝对

**没有绝对最强**：
- 没有必选战舰
- 没有必选模块
- 没有必选武器
- 不同构筑各有优势

**哲学**：
- 鼓励探索和实验
- 提高重玩价值
- 避免"唯一最优解"

### 4.4 反馈与自然教学

**即时反馈**：
- 数值提升立即生效
- 战斗反馈清晰
- 进度可见

**自然教学**：
- 通过游戏学习机制
- 不依赖冗长教程
- 新手战舰（先锋号）提供学习空间
- 危险0难度允许犯错

**哲学**：
- "Show, don't tell"
- 玩家通过玩学习
- 反馈驱动改进

### 4.5 风险与收益动态平衡

**设计原则**：
- 拾取掉落需要冒险靠近敌舰
- 高技术玩家可以更安全地拾取
- 资源回收属性提升收益，增加拾取价值
- 创造动态的风险-收益决策

**哲学**：
- Roguelike的核心平衡
- 既需要策略，也需要适应
- 每局都是独特体验

## 5. 系统协同设计

### 战舰 × 武器 × 模块

三大系统相互配合，形成独特的Build：

**示例：近战吸血流（利刃号）**
| 系统 | 选择 | 效果 |
|------|------|------|
| 战舰 | 利刃号 | 仅近程武器，能量吸收+10%，近程强化+25% |
| 武器 | 6把不同近程力场 | 高攻速多样化输出 |
| 模块 | 能量吸取器×3 + 虹吸核心 | 能量吸收+25% |
| 结果 | | 35%能量吸收+高攻速=稳定回盾 |

**示例：舰载机流（航母号）**
| 系统 | 选择 | 效果 |
|------|------|------|
| 战舰 | 航母号 | 舰载机伤害+100%，舰载指挥+30% |
| 武器 | 战斗机×2 + 无人机群×2 + 轰炸机×2 | 全舰载机 |
| 模块 | 舰载专家 + 舰载大师 + 航母指挥套装 | 舰载机伤害+80% |
| 结果 | | 舰载机总伤害+210%，本体安全 |

**示例：导弹轰炸流（炮击号）**
| 系统 | 选择 | 效果 |
|------|------|------|
| 战舰 | 炮击号 | 导弹强化+30%，爆炸范围+50% |
| 武器 | 集束导弹×2 + 追踪导弹×2 + 核弹发射器×2 | 全导弹 |
| 模块 | 导弹专家 + 导弹大师 + 穿甲弹头×2 | 导弹+40%，穿透+2 |
| 结果 | | 导弹伤害+70%，范围爆炸，穿透敌群 |

## 6. 关键数值参考

### 时间
- 单阶段战斗：20-90秒
- 补给站阶段：30-60秒
- 总游戏时长：20-30分钟
- 总阶段：20阶段

### 成长曲线
- 玩家实力：1→10倍（从第1阶段到第20阶段）
- 敌舰实力：1→10倍（匹配玩家成长）
- 资源获取：递增（后期阶段掉落更多）

### 经济
- 单阶段收入：约20-200+晶体（随阶段和资源回收属性）
- 模块价格：15-80晶体（基础价格，随阶段通胀）
- 刷新成本：2-17晶体（首次，随阶段递增）

### 难度阶段
- 阶段1-5：教学难度（虫族）
- 阶段6-10：适中难度（机械叛军）
- 阶段11-15：挑战难度（星际海盗）
- 阶段16-20：困难难度（异星帝国）

## 7. 可借鉴与创新点

### 从Brotato保留的核心
1. **清晰的核心循环**
2. **易上手难精通**
3. **精妙的数值平衡**
4. **多样化的构筑选择**
5. **策略深度**

### 我们的创新点
1. **二次元舰长养成系统**：增加情感连接和长期目标
2. **横版视角**：更适合移动端操作
3. **四大敌对势力+Boss阶段**：丰富叙事和阶段感
4. **武器类型专精**：近程/导弹/激光/舰载机四大流派
5. **好感度系统**：立绘、语音、剧情增加角色魅力
