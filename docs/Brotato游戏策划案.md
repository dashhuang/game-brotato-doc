## 1. 游戏概述

### 1.1 基本信息

| 属性 | 内容 |
|------|------|
| **游戏名称** | Brotato |
| **开发商** | Blobfish Games |
| **游戏类型** | 俯视角动作射击 + Roguelike |
| **平台** | PC (Steam)、iOS、Android |
| **发售日期** | 2022年9月27日 (EA)，2023年6月23日 (正式版) |

### 1.2 游戏简介

《Brotato》是一款快节奏的俯视角动作射击游戏，玩家扮演一名土豆士兵，需要在外星人入侵的竞技场中生存。游戏采用**自动射击**机制，玩家只需专注于**移动和躲避**，武器会自动攻击最近的敌人。

### 1.3 核心特色

- **自动射击**：降低操作门槛，专注于走位策略
- **多武器装备**：最多同时装备**6种武器**
- **丰富的Build构建**：40+角色 × 60+武器 × 150+道具
- **快节奏关卡**：每关20-90秒，一局游戏约20-30分钟
- **Roguelike元素**：随机道具、角色成长、永久解锁

## 2. 核心玩法循环

### 2.1 单局游戏流程

```
选择角色 → 选择初始武器 → 进入关卡1
     ↓
[战斗阶段] 生存指定时间，击杀敌人获取经验和材料
     ↓
[升级阶段] 使用经验值选择属性升级
     ↓
[商店阶段] 使用材料购买武器/道具/升级
     ↓
进入下一关卡 → 重复直到第20关或死亡
     ↓
结算 → 解锁新内容 → 返回主菜单
```

### 2.2 战斗阶段详解

- **持续时间**：第1关20秒，逐渐增加至第20关90秒
- **敌人刷新**：持续从屏幕边缘刷新敌人
- **Boss刷新**：特定波次会出现精英敌人/Boss
- **掉落物**：
  - 经验值宝石（蓝色）
  - 材料/金币（绿色）
  - 消耗品（生命恢复等）
  - 箱子（额外奖励）

### 2.3 升级阶段详解

每次升级可从**4个随机属性**中选择1个进行提升：

| 属性类型 | 提升内容 |
|----------|----------|
| 最大生命值 | +3 最大HP |
| 生命再生 | +1 每5秒回复 |
| 生命偷取 | +1% 攻击回血概率 |
| 伤害 | +5% 伤害加成 |
| 攻击速度 | +5% 攻速 |
| 暴击率 | +3% 暴击 |
| 护甲 | +1 护甲值 |
| 闪避 | +3% 闪避率 |

## 4. 属性系统

### 4.1 基础属性详解

#### 生存类属性

| 属性 | 说明 | 计算方式 | 上限 |
|------|------|----------|------|
| **最大生命值 (Max HP)** | 角色最大血量 | 基础10 + 加成 | 无上限 |
| **生命再生 (HP Regen)** | 每5秒回复HP量 | 数值/秒 = 属性值/5 | 无上限 |
| **生命偷取 (Lifesteal)** | 攻击回血概率 | 每次攻击 × 偷取% | 100% |
| **护甲 (Armor)** | 减少受到的伤害 | 伤害减免 = 护甲值 | 无上限 |
| **闪避 (Dodge)** | 完全躲避攻击概率 | 闪避率 = 闪避% | 60%/90%(幽灵) |

#### 攻击类属性

| 属性 | 说明 | 计算方式 | 上限 |
|------|------|----------|------|
| **伤害 (Damage)** | 基础伤害加成 | 武器伤害 × (1 + 加成%) | 无上限 |
| **攻击速度 (Attack Speed)** | 武器攻击频率 | 基础冷却 / (1 + 攻速%) | 无上限 |
| **暴击率 (Crit Chance)** | 暴击概率 | 暴击率% | 100% |
| **暴击伤害** | 暴击伤害倍率 | 默认200%，可提升 | 无上限 |
| **范围 (Range)** | 攻击/拾取范围 | 基础范围 + 加成 | 无上限 |
| **贯穿 (Piercing)** | 子弹穿透敌人数 | +1 = 多穿1个敌人 | 无上限 |

#### 特殊类属性

| 属性 | 说明 | 影响 |
|------|------|------|
| **近战伤害** | 近战武器专属加成 | 仅对近战武器生效 |
| **远程伤害** | 远程武器专属加成 | 仅对远程武器生效 |
| **元素伤害** | 元素武器专属加成 | 火/冰/电等元素武器 |
| **工程学** | 召唤物/炮塔加成 | 提升部署物威力 |
| **速度** | 角色移动速度 | 影响走位灵活性 |
| **幸运** | 稀有掉落概率 | 影响商店/掉落品质 |
| **收获** | 材料掉落加成 | 提升经济收入 |

### 4.2 属性计算公式

#### 伤害计算

```
最终伤害 = 武器基础伤害 
         × (1 + 伤害%) 
         × (1 + 类型伤害%)  // 近战/远程/元素
         × (1 + 暴击伤害%)  // 如果暴击
         - 敌人护甲
```

#### 攻击间隔计算

```
实际攻击间隔 = 武器基础间隔 / (1 + 攻速%)
```

#### 护甲减伤

```
实际受到伤害 = 原始伤害 - 护甲值
(最低伤害 = 1)
```

#### 闪避判定

```
每次被攻击时:
  随机数(0-100) < 闪避% → 完全躲避
```

## 13. 数值设计原则

### 13.1 边际效应

属性堆叠存在**边际递减**：
- 第一个+10%伤害 → 实际提升10%
- 叠加到+100%时，再+10% → 实际只提升5%

**设计建议**：均衡发展多属性，避免单一属性过度堆叠

### 13.2 阈值效应

某些属性存在**关键阈值**：
- 闪避60%（普通上限）
- 闪避90%（幽灵上限）
- 攻速达到某值可秒杀敌人

### 13.3 乘法叠加 vs 加法叠加

| 类型 | 计算方式 | 示例 |
|------|----------|------|
| 加法 | A + B | 伤害%、攻速% |
| 乘法 | (1+A) × (1+B) | 伤害×类型伤害 |

### 13.4 平衡设计

```
DPS = (武器伤害 × 伤害加成 × 类型加成) / 攻击间隔 × 暴击系数
生存时间 = 最大HP / (敌人DPS × (1-闪避) - 护甲减免 - 回复)
```

## 附录A：核心公式汇总

### 伤害公式
```
最终伤害 = 武器伤害 × (1 + 伤害%) × (1 + 类型伤害%) × 暴击倍率 - 护甲
```

### DPS公式
```
DPS = 最终伤害 / 攻击间隔 × (1 + 暴击率 × (暴击伤害 - 1))
```

### 生存公式
```
有效HP = 最大HP / (1 - 闪避) + 护甲 × 平均受击次数 + 回复
```

### 经济公式
```
材料收入 = 击杀数 × 基础掉落 × (1 + 收获%) + 波次奖励
```
