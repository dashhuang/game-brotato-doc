### 6.2 难度曲线公式（非线性，对标Brotato约10倍成长）

```
敌舰护盾 = 基础护盾 × (1 + 阶段^1.35 × 0.12) × 难度系数
敌舰伤害 = 基础伤害 × (1 + 阶段^1.25 × 0.10) × 难度系数
敌舰数量 = 基础数量 × (1 + 阶段^1.15 × 0.08) × 难度系数
敌舰速度 = 基础速度 × (1 + 阶段 × 0.03) × 难度系数
```

**倍率对照表**（难度系数=1.0时）：

| 阶段 | 护盾倍率 | 伤害倍率 | 数量倍率 | 对标Brotato |
|------|----------|----------|----------|-------------|
| 1 | 1.12x | 1.10x | 1.08x | ~1.0x |
| 5 | 2.06x | 1.74x | 1.52x | ~2.0x |
| 10 | 4.02x | 2.99x | 2.12x | ~4.0x |
| 15 | 6.72x | 4.53x | 2.78x | ~7.0x |
| 20 | 10.18x | 6.31x | 3.49x | ~10.0x |

> **设计说明**：采用指数公式确保后期难度陡峭上升。第20阶段敌人护盾约为第1阶段的10倍，对标Brotato的成长曲线。伤害成长略慢于护盾（避免后期一击秒杀玩家），数量成长最慢（避免性能问题）。

### 6.4 敌舰刷新机制

#### 刷新位置
- 敌舰从**屏幕边缘外**50-100像素处生成
- 刷新位置按**8个方向**均匀分布（上、下、左、右、四角）
- 相邻敌舰最小间距：50像素

#### 刷新时机
```
刷新间隔 = 基础间隔 / (1 + 阶段 × 0.1)
基础间隔 = 1.5秒

阶段1: 1.36秒/波
阶段10: 0.75秒/波
阶段20: 0.5秒/波
```

#### 单次刷新数量
| 阶段范围 | 基础刷新数量 | 密度描述 |
|----------|--------------|----------|
| 1-3 | 3-5只 | 低 |
| 4-5 | 5-8只 | 中 |
| 6-10 | 8-12只 | 高 |
| 11-15 | 12-18只 | 极高 |
| 16-20 | 18-25只 | 疯狂 |

#### 敌舰组合规则
- **普通刷新**：100%普通敌舰
- **精英刷新**：1只精英 + 2-4只普通敌舰
- **Boss刷新**：Boss单独出现于屏幕中央

#### 刷新上限
- 屏幕内最大敌舰数量：**100只**
- 达到上限时暂停刷新，直到击毁部分敌舰

## 七、敌舰AI行为

### 7.1 AI类型

| 类型 | 行为 | 使用敌舰 |
|------|------|----------|
| **追踪型** | 直接向玩家移动 | 孢子虫、猎杀者、劫掠者 |
| **包围型** | 尝试包围玩家 | 甲壳舰、帝国战士 |
| **远程型** | 保持距离射击 | 狙击机、离子炮舰 |
| **冲锋型** | 加速冲向玩家 | 自爆机、精英劫掠者 |
| **固定型** | 静止攻击 | 激光哨兵、炮台 |
| **游荡型** | 随机移动 | 走私船、传送舰 |
| **支援型** | 跟随其他敌舰 | 修复机、护卫舰 |

### 7.2 特殊行为

| 行为 | 描述 |
|------|------|
| **护盾再生** | 脱离战斗5秒后开始回盾 |
| **援护** | 优先攻击正在攻击友军的玩家 |
| **逃离** | 低护盾时尝试逃离 |
| **协同** | 与同类敌舰保持阵型 |

## 九、无尽深空模式

### 9.1 解锁条件
- 通关第20阶段（任意难度）

### 9.2 模式特点
- 阶段无限继续
- 难度持续增加
- 所有势力混合出现
- 定期出现随机Boss

### 9.3 难度增长

```
无尽模式敌舰属性 = 第20阶段属性 × (1 + (当前阶段-20) × 0.25)
```

### 9.4 特殊Boss
- 阶段25：随机Boss
- 阶段30：双Boss
- 阶段35+：每5阶段一个随机Boss
