# 星河舰长 舰载武器系统详解

> 本文档详细记录《星河舰长》中所有舰载武器的完整信息。

---

## 一、武器系统概述

### 1.1 武器数量
- **近程炮**：23种
- **中程导弹**：18种
- **远程激光**：15种
- **舰载机**：12种
- **总计**：68种武器

### 1.2 武器槽位
- 每艘战舰最多可同时装备**6种武器**
- 所有武器同时攻击（自动锁定最近敌舰）
- 相同武器可以装备多把

### 1.3 武器等级（Tier）

| Tier | 名称 | 颜色 | 属性倍率 | 价格倍率 |
|------|------|------|----------|----------|
| 1 | 标准型 | 白色 | ×1.0 | ×1.0 |
| 2 | 改良型 | 绿色 | ×1.25 | ×1.5 |
| 3 | 精英型 | 蓝色 | ×1.5 | ×2.0 |
| 4 | 旗舰型 | 紫色 | ×1.75 | ×2.5 |

### 1.4 武器合成机制（对标Brotato）

**核心规则**：购买已拥有的相同武器 → **2把自动合成为1把更高Tier**

| 操作 | 结果 | 武器数量变化 |
|------|------|--------------|
| 持有1把T1武器，购买第2把相同T1 | 合成为1把T2 | 2→1 |
| 持有1把T2武器，购买第2把相同T2 | 合成为1把T3 | 2→1 |
| 持有1把T3武器，购买第2把相同T3 | 合成为1把T4 | 2→1 |
| T4为最高等级 | 无法再合成 | - |

**策略考量**：
- 合成后武器数量**减少**（2把→1把）
- 需权衡："2把T2武器" vs "1把T3武器"
- 有时多把低级武器比单把高级武器更有效（武器协同效应）
- 升级后基础属性按Tier倍率提升（T2=1.25x, T3=1.5x, T4=1.75x）

| 旧像素值 | 新GU值 | 换算依据 |
|----------|--------|----------|
| 150像素 | 1.5 GU | 战舰高度≈100像素时 |
| 175像素 | 2 GU | |
| 200像素 | 2.5 GU | |
| 300像素 | 4 GU | |

> **设计说明**：通过命名让玩家直观理解武器距离：
> - 看到"力场/护盾/电弧" → 知道是最近程，需要贴身
> - 看到"脉冲/冲击" → 知道是近程，但稍有距离

## 五、舰载机 (12种)

> 特点：召唤物、自主作战、持续输出、受工程属性加成

### 5.0 舰载机系统概述 ⭐重要

舰载机是**自主作战的伴随武器**，装备后会召唤舰载机环绕在战舰周围，自动寻敌、攻击、返回。

#### 核心特性

| 特性 | 说明 | 与Brotato对比 |
|------|------|---------------|
| **自主移动** | 会飞出攻击再返回 | Brotato炮塔是静态的 |
| **锁定追杀** | 锁定目标后持续攻击直到击杀 | 更有策略感 |
| **独立攻速** | 舰载机攻速不受玩家射击频率影响 | 与Brotato一致 |
| **能量吸收生效** | 舰载机攻击**可触发能量吸收** | ⚡与Brotato不同 |
| **品级合成** | T1-T4品级，合成逻辑与普通武器相同 | 与普通武器一致 |

#### 舰载机状态机

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│   【待机】──发现敌人──→【攻击】──目标死亡──→【返回】  │
│      ↑                                      │      │
│      └──────────────────────────────────────┘      │
│                                                     │
└─────────────────────────────────────────────────────┘
```

| 状态 | 行为描述 |
|------|----------|
| **待机（IDLE）** | 环绕玩家战舰旋转，等待攻击冷却完成后寻找目标 |
| **攻击（ATTACKING）** | 飞向目标敌舰，进入攻击射程后持续开火，直到目标死亡 |
| **返回（RETURNING）** | 目标死亡后，飞回玩家身边，切换回待机状态 |

**关键设计**：无人机锁定目标后会**持续攻击直到击杀**，不会中途放弃。

#### 舰载机数量

- 每装备1把舰载机武器 = 召唤1架舰载机
- 最多6架同时环绕（均匀分布）
- 合成后数量减少（2把T1 → 1把T2 = 2架变1架）

### 无人机类 ⭐详细设计

#### 无人机核心参数

| 参数 | 说明 |
|------|------|
| **寻敌范围（seekRange）** | 待机时检测以玩家为中心该范围内的敌人 |
| **攻击射程（attackRange）** | 无人机需飞到距离目标该距离以内才会开火 |
| **出击速度（flySpeed）** | 飞向敌人的速度 |
| **返回速度（returnSpeed）** | 返回玩家身边的速度 |
| **出击冷却** | 返回后进入待机，冷却完成后才会再次出击 |

## 6.1 武器命中机制（单体 vs AOE）⭐重要

> **对标Brotato**：明确定义武器的命中目标数量机制

### 6.1.0 核心规则

| 武器大类 | 默认命中 | 说明 | Brotato对应 |
|----------|----------|------|-------------|
| **近程武器** | **AOE（多体）** | 攻击范围内的所有敌舰都会被命中 | 近战武器天然AOE |
| **导弹武器** | **单体** | 每发导弹只命中1个敌舰 | 远程武器天然单体 |
| **激光武器** | **单体** | 每道激光只命中1个敌舰（除非穿透） | 远程武器天然单体 |
| **舰载机** | **单体** | 每次攻击只命中1个敌舰 | 召唤物单体 |

### 6.1.1 近程武器的AOE机制

近程武器（力场/电弧/脉冲/近防）**天然是AOE**：

```
攻击触发 → 检测射程范围内所有敌舰 → 全部造成伤害
```

| 近程类型 | 射程 | AOE方式 | 视觉表现 |
|----------|------|---------|----------|
| **力场系列** | 1.5 GU | 环绕能量场 | 360°能量波动 |
| **电弧系列** | 2 GU | 电弧传导 | 闪电弧光连接多目标 |
| **脉冲系列** | 2.5 GU | 冲击波扩散 | 圆形冲击波 |
| **近防系列** | 3-4 GU | 扇形射击 | 多道子弹扇形散射 |

**设计意义**：
- 近程武器射程短，用AOE弥补
- 鼓励玩家冒险贴近敌群
- 配合能量吸收（攻击回盾）形成高风险高回报

### 6.1.2 远程武器变AOE的方法

导弹和激光默认单体，可通过以下方式变成AOE：

#### 方法1：穿透（Piercing）

| 武器/效果 | 穿透数量 | 说明 |
|-----------|----------|------|
| 激光狙击 | 无限 | 贯穿全屏所有敌舰 |
| 穿甲鱼雷 | 3 | 穿透3个敌舰 |
| 穿透属性+1 | +1目标 | 每点穿透多命中1个 |

```
弹道发射 → 命中敌舰A → 穿透继续飞行 → 命中敌舰B → ...
```

#### 方法2：爆炸（Explosion）

| 武器 | 爆炸半径 | 说明 |
|------|----------|------|
| 集束导弹 | 2 GU | 爆炸范围内全部受伤 |
| 战术核弹 | 4 GU | 大范围爆炸 |
| 轰炸无人机 | 1.5 GU | 爆炸攻击 |

```
弹道命中 → 产生爆炸 → 爆炸范围内所有敌舰受伤
```

#### 方法3：连锁（Chain）

| 武器 | 连锁次数 | 连锁范围 |
|------|----------|----------|
| 离子连锁炮 | 3次 | 3 GU |
| 连锁电弧 | 5次 | 4 GU |

```
命中敌舰A → 跳跃到3GU内敌舰B → 跳跃到敌舰C → ...
```

#### 方法4：散射（Multi-projectile）

| 武器 | 子弹数 | 说明 |
|------|--------|------|
| 集束导弹（发射时） | 6发 | 每发独立命中 |
| 散射炮 | 5发 | 扇形散射 |

```
一次攻击 → 发射多发弹道 → 每发独立命中1个敌舰
```

### 6.1.3 命中机制与属性的交互

| 属性/效果 | 单体武器 | AOE武器 |
|-----------|----------|---------|
| **能量吸收** | 每次命中1次触发 | 每个被命中敌舰各触发1次 |
| **击毁回盾** | 击毁1个回复 | 同时击毁多个各自回复 |
| **DOT效果** | 附加到1个敌舰 | 附加到所有被命中敌舰 |
| **击退效果** | 击退1个敌舰 | 击退所有被命中敌舰 |

**关键结论**：
- AOE武器配合"能量吸收"更有效（多次触发）
- AOE武器配合"DOT效果"更有效（多目标附加）

---

### 6.1.4 伤害类型

| 类型 | 加成来源 | 武器示例 |
|------|----------|----------|
| **近程伤害** | 近程强化% | 速射炮、冲击波发生器 |
| **导弹伤害** | 导弹强化% | 追踪导弹、鱼雷 |
| **能量伤害** | 能量武器% | 激光、等离子 |
| **召唤物伤害** | 工程% | 炮塔、无人机、战机、空间站 |

### 6.2 伤害计算公式

```
武器最终伤害 = 基础伤害 
             × Tier倍率 
             × (1 + 火力输出%) 
             × (1 + 武器类型加成%)
             × (1 + 致命伤害%) [如果暴击]
```

### 6.3 DPS计算

```
武器DPS = 最终伤害 × 子弹数 / 冷却时间 × (1 + 射击频率%)
        × (1 + 精确打击率 × (致命伤害 - 1))
```
