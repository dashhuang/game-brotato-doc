# 星河舰长 舰载武器系统详解

> 本文档详细记录《星河舰长》中所有舰载武器的完整信息。

---

## 一、武器系统概述

### 1.1 武器数量
- **近程炮**：23种
- **中程导弹**：18种
- **远程激光**：15种
- **舰载机**：12种
- **总计**：68种武器

### 1.2 武器槽位
- 每艘战舰最多可同时装备**6种武器**
- 所有武器同时攻击（自动锁定最近敌舰）
- 相同武器可以装备多把

### 1.3 武器等级（Tier）

| Tier | 名称 | 颜色 | 属性倍率 | 价格倍率 |
|------|------|------|----------|----------|
| 1 | 标准型 | 白色 | ×1.0 | ×1.0 |
| 2 | 改良型 | 绿色 | ×1.25 | ×1.5 |
| 3 | 精英型 | 蓝色 | ×1.5 | ×2.0 |
| 4 | 旗舰型 | 紫色 | ×1.75 | ×2.5 |

### 1.4 武器合成机制（对标Brotato）

**核心规则**：购买已拥有的相同武器 → **2把自动合成为1把更高Tier**

| 操作 | 结果 | 武器数量变化 |
|------|------|--------------|
| 持有1把T1武器，购买第2把相同T1 | 合成为1把T2 | 2→1 |
| 持有1把T2武器，购买第2把相同T2 | 合成为1把T3 | 2→1 |
| 持有1把T3武器，购买第2把相同T3 | 合成为1把T4 | 2→1 |
| T4为最高等级 | 无法再合成 | - |

**策略考量**：
- 合成后武器数量**减少**（2把→1把）
- 需权衡："2把T2武器" vs "1把T3武器"
- 有时多把低级武器比单把高级武器更有效（武器协同效应）
- 升级后基础属性按Tier倍率提升（T2=1.25x, T3=1.5x, T4=1.75x）

## 二、近程武器 (22种)

> 特点：高射速、无限穿透、射程150-300、受近程强化加成

### 射程分类说明（命名暗示距离）

| 射程 | 武器类型 | 核心概念 | 视觉特征 | 玩家理解 |
|------|----------|----------|----------|----------|
| **150** | **力场/护盾系列** | 环绕舰体的能量场 | 能量波动、护盾延伸 | "场"=环绕=贴身 |
| **175** | **电弧/放电系列** | 电弧传导攻击 | 闪电弧光、放电效果 | 电弧=传导=近距离 |
| **200** | **脉冲/冲击系列** | 冲击波扩散 | 脉冲波动、冲击圈 | 冲击波=扩散=短程 |
| **225-300** | **近防/突击系列** | 近程防御火力 | 炮塔射击、主炮开火 | 直接说明近程 |

> **设计说明**：通过命名让玩家直观理解武器距离：
> - 看到"力场/护盾/电弧" → 知道是最近程，需要贴身
> - 看到"脉冲/冲击" → 知道是近程，但稍有距离
> - 看到"近防/突击" → 知道是近程火力

## 六、武器属性详解

### 6.1 伤害类型

| 类型 | 加成来源 | 武器示例 |
|------|----------|----------|
| **近程伤害** | 近程强化% | 速射炮、冲击波发生器 |
| **导弹伤害** | 导弹强化% | 追踪导弹、鱼雷 |
| **能量伤害** | 能量武器% | 激光、等离子 |
| **舰载机伤害** | 舰载指挥% | 炮塔、无人机、战机 |

### 6.2 伤害计算公式

```
武器最终伤害 = 基础伤害 
             × Tier倍率 
             × (1 + 火力输出%) 
             × (1 + 武器类型加成%)
             × (1 + 致命伤害%) [如果暴击]
```

### 6.3 DPS计算

```
武器DPS = 最终伤害 × 子弹数 / 冷却时间 × (1 + 射击频率%)
        × (1 + 精确打击率 × (致命伤害 - 1))
```

## 八、武器获取方式

### 8.1 补给站购买
- 每波结束后补给站刷新4个物品
- 可花费能量晶体刷新补给站
- 物品品质受战场运势属性影响

### 8.2 初始武器
- 大部分战舰有1把初始武器
- 部分战舰有2把或无初始武器
- 初始武器为Tier 1

### 8.3 补给箱掉落
- 战场中随机出现补给箱
- 补给箱可能掉落武器
- 掉落品质随机
