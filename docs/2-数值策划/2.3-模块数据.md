# 星河舰长 舰船模块系统详解

> 本文档详细记录《星河舰长》中所有舰船模块的完整信息。

## 一、模块系统概述

### 1.1 模块数量
- **火控模块**：35种
- **防御模块**：28种
- **引擎模块**：25种
- **战术模块**：22种
- **实验模块**：15种
- **特殊模块**：10种
- **总计**：约144种模块

### 1.2 模块特点
- 模块为**被动装备**，装备后持续生效
- 模块**无装备上限**，可以无限叠加（可叠加模块）
- 相同模块可以多次购买，效果叠加
- 模块无等级系统，价格固定

### 1.3 唯一 vs 可叠加模块 ⭐（对标Brotato核心设计）

> **设计理念**：某些强力模块设为"唯一"，购买后不再出现在补给站，防止无限叠加破坏平衡。

| 类型 | 说明 | 设计目的 |
|------|------|----------|
| **可叠加** ✓ | 可多次购买，效果线性叠加 | 允许Build专精，数值逐步提升 |
| **唯一** ⚡ | 购买后不再出现在补给站 | 防止滥用，增加选择重要性 |
