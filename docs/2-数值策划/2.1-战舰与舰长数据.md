# 星河舰长 战舰与舰长系统详解

> 本文档详细记录《星河舰长》中所有战舰和舰长的完整信息。

---

## 一、战舰系统概述

### 1.1 战舰数量
- **突击舰**：10艘
- **火力舰**：8艘
- **防御舰**：8艘
- **特种舰**：8艘
- **支援舰**：6艘
- **旗舰**：8艘
- **总计**：48艘战舰（对应48位舰长）

### 1.2 战舰属性结构

每艘战舰包含：
- **舰型名称**：战舰的型号名
- **舰长姓名**：对应舰长名称
- **初始武器**：开局自带的武器
- **初始模块**：开局自带的模块（部分战舰有）
- **属性加成**：正面效果
- **属性减益**：负面效果 ⭐
- **特殊能力**：独特的被动技能
- **解锁条件**：如何解锁该战舰

### 1.2.1 正负属性设计理念 ⭐（对标Brotato核心设计）

> **设计哲学**：强力的正面属性必须伴随明显的负面属性，这是创造策略深度和避免"必选战舰"的关键。

#### 设计原则

| 原则 | 说明 | 示例 |
|------|------|------|
| **代价平衡** | 正面效果越强，负面效果越重 | 狙击号：+100%伤害 / -60%攻速 |
| **塑造特色** | 负面属性定义战舰独特玩法 | 利刃号：仅近程武器 |
| **避免必选** | 没有"纯正面"的强力战舰 | 先锋号：无负面，但加成也较弱 |
| **鼓励多样** | 不同负面催生不同Build | 导弹流 vs 近程流 |

#### 负面属性类型

| 负面类型 | 效果 | 应对策略 | 代表战舰 |
|----------|------|----------|----------|
| **属性削弱** | 降低某项属性 | 通过模块弥补 | 疾风号：静止时装甲-5 |
| **武器限制** | 只能使用某类武器 | 专精该类武器 | 毁灭号：仅导弹类武器 |
| **机制改变** | 改变游戏规则 | 适应新玩法 | 吸血号：无法使用修复包 |
| **风险增加** | 敌人更强或自伤 | 需要更高技术 | 喧嚣号：敌舰+20%伤害 |

#### 负面属性强度分级

| 强度 | 描述 | 示例 |
|------|------|------|
| ⚪ 轻微 | 几乎无影响 | 引擎推力-5% |
| 🔵 中等 | 需要适应 | 最大护盾-25% |
| 🟣 较重 | 改变策略 | 无法使用修复包 |
| 🔴 严重 | 核心限制 | 仅能装备1武器 |

#### 战舰设计示例解析

| 战舰 | 正面效果 | 负面效果 | 设计意图 |
|------|----------|----------|----------|
| **利刃号** | 能量吸收+10%，近程强化+25% | 导弹强化-100%（仅近程） | 定义近战专精玩法 |
| **狙击号** | 火力+100%，范围+200，穿透+5 | 攻速-60%，速度-20% | 超高单发伤害，需精准走位 |
| **单舰号** | 全属性+25%，武器伤害+100% | 只能装备1武器，护盾=1 | 极限挑战，单武器精通 |
| **恶魔号** | 击毁回复1护盾，火力+30% | 每秒损失1护盾 | 必须持续击杀才能生存 |

#### 新手友好 vs 挑战战舰

| 类型 | 特点 | 代表战舰 |
|------|------|----------|
| **新手友好** | 无/轻微负面，容错率高 | 先锋号、猎鹰号、重巡号 |
| **进阶** | 中等负面，需要理解机制 | 利刃号、毁灭号、工程号 |
| **挑战** | 严重负面，需要高技术 | 单舰号、恶魔号、和平号 |

### 1.3 战舰基础数值（所有战舰统一）

| 属性 | 基础值 | 说明 |
|------|--------|------|
| **最大护盾** | 25 | 可通过模块/升级提升（对标Brotato初始生命20-30） |
| **护盾再生** | 0 | 每5秒回复量 |
| **能量吸收** | 0% | 攻击回复护盾概率 |
| **装甲值** | 0 | 减少受到伤害 |
| **规避率** | 0% | 完全闪避概率，上限60% |
| **火力输出** | 0% | 伤害加成 |
| **射击频率** | 0% | 攻速加成 |
| **精确打击** | 0% | 暴击概率 |
| **致命伤害** | 100% | 暴击额外伤害 |
| **火力范围** | 0 | 额外攻击范围 |
| **引擎推力** | 100% | 移动速度百分比 |
| **战场运势** | 0 | 影响掉落品质 |
| **资源回收** | 0 | 能量晶体额外获取 |

| **拾取范围** | 150 | 自动拾取掉落物范围 |

> **设计说明**：基础护盾25点确保玩家在第1阶段（敌人伤害1-4）有足够容错空间，可承受6-25次攻击。对标Brotato"全能者"角色初始生命值约20-30。
