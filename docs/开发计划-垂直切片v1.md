# 星河舰长 - 垂直切片开发计划 v1

> **目标**：用最小可行内容验证核心玩法，快速迭代，确认游戏的核心乐趣

---

## 📋 垂直切片概述

### 什么是垂直切片？

垂直切片是一个**功能完整但内容精简**的可玩版本：
- ✅ 所有核心系统都能运行
- ✅ 一局完整游戏可以体验
- ✅ 不同Build构建有区分度
- ❌ 不追求内容丰富
- ❌ 不追求美术精美

### 验证目标

| 目标 | 验证内容 | 成功标准 |
|------|----------|----------|
| **核心循环** | 战斗→升级→补给站→战斗 | 循环流畅，无卡顿感 |
| **多武器乐趣** | 6把武器同时射击 | 视觉爽感，策略深度 |
| **经济系统** | 收获→战斗的投资节奏 | 前期投资有回报，后期有压力 |
| **Build多样性** | 不同武器/模块组合 | 至少3种有效流派 |
| **难度曲线** | 10/20波的挑战感 | 有难度但不绝望 |

## ✅ 验收标准

### 核心功能验收

- [ ] 能完成完整10波战斗
- [ ] 5艘战舰均可选择并有明显体验差异
- [ ] 15种武器均可正常使用
- [ ] 25种模块效果正确生效
- [ ] 补给站系统完整可用
- [ ] Boss战可以挑战并击败

### 体验验收

- [ ] 至少3种不同Build可以通关
- [ ] 经济压力感知明确（前期投资vs后期购买）
- [ ] 难度曲线感知明确（前期轻松→后期紧张）
- [ ] 多武器射击有视觉爽感
- [ ] 单局时长控制在10-15分钟

### 数值验收

- [ ] 玩家有30%左右的失败率
- [ ] 不同战舰通关率差异<20%
- [ ] 没有明显的OP武器/模块
- [ ] 经济节奏符合"收获→战斗"转型
