# Brotato 控制与UI设计详解

> 本文档详细记录《Brotato》的操作方式、UI界面设计和用户体验要素。

---

## 一、控制系统

### 1.1 核心控制理念

《Brotato》采用**极简控制**设计：
- **自动射击**：无需手动瞄准和攻击
- **只需移动**：玩家只控制角色移动
- **降低门槛**：专注于策略和走位

### 1.2 PC端控制

| 操作 | 键位 | 说明 |
|------|------|------|
| **移动** | WASD / 方向键 | 八方向移动 |
| **暂停** | ESC / P | 暂停游戏 |
| **地图查看** | Tab / M | 查看全图（如有） |

### 1.3 手柄控制

| 操作 | 键位 | 说明 |
|------|------|------|
| **移动** | 左摇杆 | 360度移动 |
| **暂停** | Start | 暂停游戏 |
| **确认** | A / X | 商店确认 |
| **取消** | B / O | 返回 |

### 1.4 移动端控制

| 操作 | 手势 | 说明 |
|------|------|------|
| **移动** | 虚拟摇杆 | 左下角虚拟摇杆 |
| **暂停** | 右上角按钮 | 暂停图标 |
| **商店操作** | 点击 | 直接点击物品 |

### 1.5 移动速度计算

```
实际速度 = 基础速度 × (1 + 速度加成%)
基础速度 = 200像素/秒
```

## 六、视觉反馈

### 6.1 战斗特效

| 事件 | 视觉效果 |
|------|----------|
| 武器攻击 | 枪口闪光/挥砍轨迹 |
| 命中敌人 | 冲击波纹/血液飞溅 |
| 暴击 | 更大的特效+屏幕轻微震动 |
| 击杀敌人 | 敌人消失动画+掉落物飞出 |
| 燃烧效果 | 敌人身上火焰特效 |
| 冰冻效果 | 敌人变蓝+减速线 |
| 闪电连锁 | 敌人间闪电弧线 |

### 6.2 角色状态反馈

| 状态 | 视觉效果 |
|------|----------|
| 受伤 | 屏幕边缘闪红 |
| 低血量 | 屏幕持续红边+心跳音效 |
| 升级 | 角色发光+短暂减速 |
| 获得道具 | 道具图标飘向角色 |
| 闪避成功 | 角色闪烁 |

### 6.3 敌人状态反馈

| 状态 | 视觉效果 |
|------|----------|
| 精英敌人 | 黄色轮廓 |
| Boss | 红色轮廓+血条 |
| 燃烧中 | 身上火焰 |
| 中毒中 | 身上绿色气泡 |
| 冰冻中 | 蓝色外观 |
| 低血量 | 闪烁 |
