# 星河舰长 舰长养成系统详解

> 本文档详细记录《星河舰长》中舰长养成的完整机制，包括好感度、语音、剧情等系统。

---

## 一、舰长系统概述

### 1.1 设计理念

舰长是《星河舰长》区别于传统Roguelike的核心特色：
- 每艘战舰对应1位女性舰长
- 舰长提供**情感连接**和**收集动力**
- 舰长养成是**永久进度**的主要载体
- 舰长不直接参与战斗，但提供语音反馈和剧情

### 1.2 舰长数量

| 类型 | 数量 |
|------|------|
| 突击舰舰长 | 10人 |
| 火力舰舰长 | 8人 |
| 防御舰舰长 | 8人 |
| 特种舰舰长 | 8人 |
| 支援舰舰长 | 6人 |
| 旗舰舰长 | 8人 |
| **总计** | **48人** |

## 二、好感度系统

### 2.1 好感度等级

| 等级 | 名称 | 所需点数 | 累计点数 | 解锁内容 |
|------|------|----------|----------|----------|
| 0 | 陌生人 | - | 0 | 基础立绘、基础语音 |
| 1 | 战友 | 100 | 100 | 剧情第1章、档案开放 |
| 2 | 同伴 | 200 | 300 | 语音包1、剧情第2章 |
| 3 | 挚友 | 300 | 600 | 皮肤1、剧情第3章 |
| 4 | 知己 | 400 | 1000 | 语音包2、剧情第4章 |
| 5 | 誓约 | 500 | 1500 | 誓约皮肤、完整剧情、特殊效果 |

### 2.2 好感度获取方式

| 方式 | 获取量 | 条件 |
|------|--------|------|
| 使用该战舰通关 | +20 | 通关危险0 |
| 使用该战舰通关危险1 | +25 | 通关危险1 |
| 使用该战舰通关危险2 | +30 | 通关危险2 |
| 使用该战舰通关危险3 | +35 | 通关危险3 |
| 使用该战舰通关危险4 | +40 | 通关危险4 |
| 使用该战舰通关危险5 | +50 | 通关危险5 |
| 首次解锁该战舰 | +50 | 一次性 |
| 首次使用该战舰 | +10 | 一次性 |
| 无伤通关 | +10 (额外) | 全程未受伤 |
| 完成舰长特定挑战 | +20-50 | 每舰长独特 |

### 2.3 舰长特定挑战

每位舰长有3个特定挑战，完成后获得额外好感度：

| 挑战类型 | 示例 | 奖励 |
|----------|------|------|
| **战斗挑战** | 使用该战舰击毁10000敌舰 | +50 |
| **通关挑战** | 使用该战舰通关危险5 | +30 |
| **特殊挑战** | 使用该战舰单武器通关 | +20 |

## 四、剧情系统

### 4.1 剧情结构

每位舰长拥有**5章个人剧情**：

| 章节 | 名称 | 解锁条件 | 内容 |
|------|------|----------|------|
| 第1章 | 初见 | 好感度等级1 | 舰长的背景故事和初次相遇 |
| 第2章 | 相知 | 好感度等级2 | 舰长的过去和成长经历 |
| 第3章 | 信任 | 好感度等级3 | 舰长的秘密和内心世界 |
| 第4章 | 羁绊 | 好感度等级4 | 舰长与玩家的特殊经历 |
| 第5章 | 誓约 | 好感度等级5 | 舰长的誓约和未来 |

### 4.2 剧情呈现方式

- **Visual Novel风格**：立绘+对话框+背景
- **表情差分**：根据对话内容切换表情
- **CG插画**：关键剧情节点有专属CG
- **语音配音**：全剧情语音(可选)
- **选择分支**：部分对话有选择，影响细节

### 4.3 剧情奖励

| 章节 | 奖励 |
|------|------|
| 完成第1章 | 解锁档案详情 |
| 完成第2章 | 获得10能量晶体(永久) |
| 完成第3章 | 解锁战舰皮肤 |
| 完成第4章 | 获得专属称号 |
| 完成第5章 | 解锁誓约皮肤、战斗特效 |

## 五、立绘系统

| 类型 | 用途 | 规格建议 |

## 七、誓约系统

### 7.1 誓约条件

| 条件 | 要求 |
|------|------|
| 好感度等级 | 5级 |
| 剧情完成 | 第5章完成 |
| 特殊道具 | 誓约戒指(通关奖励获得) |

### 7.2 誓约效果

| 效果 | 描述 |
|------|------|
| **誓约皮肤** | 解锁专属誓约立绘 |
| **战斗特效** | 使用该战舰时有专属特效 |
| **专属语音** | 誓约后专属语音包 |
| **称号** | "XX的誓约者"称号 |
| **战斗加成** | 使用该战舰时全属性+1% |

### 7.3 誓约仪式

- 播放专属誓约动画
- 舰长说出誓约台词
- 交换誓约戒指
- 解锁誓约纪念日

## 八、母港系统

### 8.1 母港功能

| 功能 | 说明 |
|------|------|
| **舰长展示** | 展示已解锁舰长立绘 |
| **好感度查看** | 查看所有舰长好感度进度 |
| **剧情回放** | 重新观看已解锁剧情 |
| **语音鉴赏** | 收听已解锁语音 |
| **CG鉴赏** | 查看已解锁CG |
| **档案查阅** | 查看舰长档案 |

### 8.2 母港互动

- **点击舰长**：触发随机语音
- **特殊日期**：生日/节日触发特殊语音
- **长时间未游戏**：回归时触发欢迎语音
- **成就达成**：舰长祝贺语音

## 十、舰长养成建议

### 10.1 效率养成

| 目标 | 策略 |
|------|------|
| 快速满好感 | 专注一个舰长，高难度通关 |
| 广泛解锁 | 轮流使用不同战舰 |
| 完美收集 | 完成所有舰长挑战 |

### 10.2 预期进度

| 游戏时长 | 预期进度 |
|----------|----------|
| 10小时 | 2-3个舰长满好感 |
| 50小时 | 10-15个舰长满好感 |
| 100小时 | 30+舰长满好感 |
| 200小时 | 全舰长满好感 |
