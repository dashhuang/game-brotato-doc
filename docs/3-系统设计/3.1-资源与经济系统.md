# 星河舰长 资源与补给系统详解

> 本文档详细记录《星河舰长》中的资源获取、补给站购买和经济机制。

---

## 一、资源类型

### 1.1 能量晶体 (Energy Crystal)

| 属性 | 描述 |
|------|------|
| **图标** | 💎 绿色六边形结晶 |
| **用途** | **双重功能**：积累升级 + 购买武器/模块 |
| **持久性** | 仅当局有效 |

> **设计说明**：能量晶体同时充当"经验值"和"货币"两种角色。收集足够晶体后触发舰桥升级，同时晶体也用于补给站购买。这意味着玩家需要在"升级属性"和"购买装备"之间做出权衡。

### 1.2 好感度点数 (Affection Points)

| 属性 | 描述 |
|------|------|
| **图标** | 💗 粉色爱心 |
| **用途** | 舰长养成 |
| **持久性** | 永久保存 |
| **获取时机** | 战斗结算后 |


## 三、舰桥升级系统

### 3.1 升级机制

能量晶体在用于购买的同时，也会**累计升级进度**。当收集的晶体达到升级阈值时，触发舰桥升级。

> **重要**：升级消耗的是"累计收集量"，不是"当前持有量"。购买物品不影响升级进度。

## 四、补给站系统

### 4.1 补给站界面

每个战役阶段结束后进入补给站，包含：
- **物品区域**：4个随机物品（武器或模块）
- **刷新按钮**：花费晶体刷新物品
- **锁定按钮**：锁定物品不被刷新
- **战舰信息**：当前属性面板
- **继续按钮**：进入下一阶段

### 4.2 物品刷新机制

#### 基础刷新规则
- 每次显示**4个物品**
- 物品可能是武器或模块
- 已拥有的武器显示**"合成"**标记（购买后2把合成为1把更高Tier）

#### 刷新费用（指数递增）

| 刷新次数 | 费用 | 累计费用 |
|----------|------|----------|
| 第1次 | 5 晶体 | 5 |
| 第2次 | 10 晶体 | 15 |
| 第3次 | 20 晶体 | 35 |
| 第4次 | 40 晶体 | 75 |
| 第5次 | 80 晶体 | 155 |
| 第N次 | 5 × 2^(N-1) 晶体 | - |

*注：每个阶段刷新次数重置*

> **设计说明**：指数递增是Brotato核心经济约束机制。线性增长（如每次+2）会让玩家可以无限刷新直到找到最优物品，破坏决策张力。指数递增确保玩家在第3-4次刷新后就需要权衡"是否值得继续刷新"，增加策略深度。

### 4.3 物品稀有度概率

#### 基础概率（无运势加成）

| 稀有度 | Tier | 概率 |
|--------|------|------|
| 标准型 | 1 | 55% |
| 改良型 | 2 | 30% |
| 精英型 | 3 | 12% |
| 旗舰型 | 4 | 3% |

#### 战场运势影响

```
标准型概率 = 55% - (运势 × 0.3%)
改良型概率 = 30% + (运势 × 0.15%)
精英型概率 = 12% + (运势 × 0.1%)
旗舰型概率 = 3% + (运势 × 0.05%)
```

**示例**（运势+50）：
- 标准型：55% - 15% = 40%
- 改良型：30% + 7.5% = 37.5%
- 精英型：12% + 5% = 17%
- 旗舰型：3% + 2.5% = 5.5%

### 4.4 物品锁定

- 每个物品可单独锁定
- 锁定的物品在刷新时保留
- 锁定**不消耗**晶体
- 锁定的物品在下一阶段**不会保留**

### 4.5 武器合成（对标Brotato）

**机制**：购买已拥有的相同武器 → **2把自动合成为1把更高Tier**

| 情况 | 操作 | 结果 |
|------|------|------|
| 持有1把T1速射脉冲 | 购买第2把T1速射脉冲 | 2把合成为1把T2速射脉冲 |
| 持有1把T2速射脉冲 | 购买第2把T2速射脉冲 | 2把合成为1把T3速射脉冲 |
| 持有1把T4武器 | 无法再合成 | 提示"已达最高等级" |

**注意**：
- 合成后武器数量**减少**（从2把变为1把）
- 合成价格 = 第2把武器的正常价格
- 需权衡"多把低级"vs"单把高级"的策略价值

## 七、经济策略（核心：收获→战斗转型）

> **⚠️ Brotato核心策略**：前期投资"资源回收"滚雪球，中期逐步转型，后期全力战斗属性。这是Roguelike经济的精髓！

### 7.0 投资转型节奏图（对标Brotato）

```
阶段    资源回收投资    战斗属性投资    策略重心
────────────────────────────────────────────────────
1-5     ████████░░      ██░░░░░░░░      【滚雪球期】重点资源回收
6-10    ██████░░░░      ████░░░░░░      【转型期】平衡发展
11-15   ██░░░░░░░░      ████████░░      【成型期】减少回收，强化战斗
16-20   ░░░░░░░░░░      ██████████      【冲刺期】完全放弃回收
```

### 7.1 滚雪球期 (阶段1-5) ⭐ 资源回收优先

| 策略 | 说明 | 优先级 |
|------|------|--------|
| **资源回收** | 每次舰桥升级优先选+资源回收 | ★★★★★ |
| **最大护盾** | 保证基础生存 | ★★★★☆ |
| **武器填槽** | 买便宜T1武器填满6槽 | ★★★☆☆ |
| **避免** | 过度刷新、购买昂贵模块 | - |

**目标**：资源回收达到+30~50%，为后续积累经济优势

### 7.2 转型期 (阶段6-10) ⚖️ 平衡发展

| 策略 | 说明 | 优先级 |
|------|------|--------|
| **明确Build方向** | 确定近程/导弹/激光/舰载机流派 | ★★★★★ |
| **资源回收** | 继续提升，但不再是唯一重点 | ★★★☆☆ |
| **核心武器** | 开始合成核心武器到T2/T3 | ★★★★☆ |
| **战斗属性** | 开始投资火力/攻速/暴击 | ★★★★☆ |

**目标**：资源回收达到+50~80%，同时Build初步成型

### 7.3 成型期 (阶段11-15) ⚔️ 战斗优先

| 策略 | 说明 | 优先级 |
|------|------|--------|
| **战斗属性** | 火力/攻速/暴击/专精加成 | ★★★★★ |
| **生存属性** | 护盾/装甲/能量吸收 | ★★★★☆ |
| **资源回收** | 不再投资，维持现有 | ★☆☆☆☆ |
| **武器合成** | 核心武器升到T3/T4 | ★★★★☆ |

**目标**：Build完全成型，战斗属性达到中高水平

### 7.4 冲刺期 (阶段16-20) 🔥 极限强化

| 策略 | 说明 | 优先级 |
|------|------|--------|
| **战斗属性** | 极限堆叠核心属性 | ★★★★★ |
| **生存属性** | 补足短板 | ★★★★★ |

| **刷新** | 允许多次刷新寻找关键模块 | ★★★☆☆ |

**目标**：应对最强敌人，完成通关

## 八、难度对经济的影响

### 8.1 难度与掉落

| 危险等级 | 掉落倍率 | 敌舰数量 |
|----------|----------|----------|
| 0 | ×0.8 | ×0.7 |
| 1 | ×1.0 | ×1.0 |
| 2 | ×1.1 | ×1.15 |
| 3 | ×1.2 | ×1.3 |
| 4 | ×1.3 | ×1.5 |
| 5 | ×1.4 | ×1.75 |

### 8.2 经济平衡

高难度下：
- 敌舰更多 → 更多掉落机会
- 但战斗更危险 → 需要更强装备
- 整体收益略高，但需要更快决策
