# 星河舰长 舰载武器系统详解

> 本文档详细记录《星河舰长》中所有舰载武器的完整信息。

---

## 一、武器系统概述

### 1.1 武器数量
- **近程炮**：22种
- **中程导弹**：18种
- **远程激光**：15种
- **舰载机**：12种
- **总计**：67种武器

### 1.2 武器槽位
- 每艘战舰最多可同时装备**6种武器**
- 所有武器同时攻击（自动锁定最近敌舰）
- 相同武器可以装备多把

### 1.3 武器等级（Tier）

| Tier | 名称 | 颜色 | 属性倍率 | 价格倍率 |
|------|------|------|----------|----------|
| 1 | 标准型 | 白色 | ×1.0 | ×1.0 |
| 2 | 改良型 | 绿色 | ×1.25 | ×1.5 |
| 3 | 精英型 | 蓝色 | ×1.5 | ×2.0 |
| 4 | 旗舰型 | 紫色 | ×1.75 | ×2.5 |

### 1.4 武器升级机制
- 在补给站购买**已拥有的相同武器**会升级
- 升级将武器提升1个Tier
- Tier 4为最高等级
- 升级后基础属性按倍率提升

## 六、武器属性详解

### 6.1 伤害类型

| 类型 | 加成来源 | 武器示例 |
|------|----------|----------|
| **近程伤害** | 近程强化% | 速射炮、电磁鞭 |
| **导弹伤害** | 导弹强化% | 追踪导弹、鱼雷 |
| **能量伤害** | 能量武器% | 激光、等离子 |
| **舰载机伤害** | 舰载指挥% | 炮塔、无人机、战机 |

### 6.2 伤害计算公式

```
武器最终伤害 = 基础伤害 
             × Tier倍率 
             × (1 + 火力输出%) 
             × (1 + 武器类型加成%)
             × (1 + 致命伤害%) [如果暴击]
```

### 6.3 DPS计算

```
武器DPS = 最终伤害 × 子弹数 / 冷却时间 × (1 + 射击频率%)
        × (1 + 精确打击率 × (致命伤害 - 1))
```

## 八、武器获取方式

### 8.1 补给站购买
- 每波结束后补给站刷新4个物品
- 可花费能量晶体刷新补给站
- 物品品质受战场运势属性影响

### 8.2 初始武器
- 大部分战舰有1把初始武器
- 部分战舰有2把或无初始武器
- 初始武器为Tier 1

### 8.3 补给箱掉落
- 战场中随机出现补给箱
- 补给箱可能掉落武器
- 掉落品质随机

## 九、武器解锁条件

大部分武器默认解锁，以下武器需要满足条件：

| 武器 | 解锁条件 |
|------|----------|
| 幽能匕首 | 击毁1000敌舰 |
| 幽能战斧 | 使用粒子炮击毁500敌舰 |
| 核弹 | 造成50000爆炸伤害 |
| 激光狙击 | 使用重型鱼雷击毁200敌舰 |
| 重型战机 | 使用舰载机击毁1000敌舰 |
