# 星河舰长 战舰与舰长系统详解

> 本文档详细记录《星河舰长》中所有战舰和舰长的完整信息。

---

## 一、战舰系统概述

### 1.1 战舰数量
- **突击舰**：10艘
- **火力舰**：8艘
- **防御舰**：8艘
- **特种舰**：8艘
- **支援舰**：6艘
- **旗舰**：8艘
- **总计**：48艘战舰（对应48位舰长）

### 1.2 战舰属性结构

每艘战舰包含：
- **舰型名称**：战舰的型号名
- **舰长姓名**：对应舰长名称
- **初始武器**：开局自带的武器
- **初始模块**：开局自带的模块（部分战舰有）
- **属性加成**：正面效果
- **属性减益**：负面效果
- **特殊能力**：独特的被动技能
- **解锁条件**：如何解锁该战舰
