# Brotato 经济与商店系统详解

> 本文档详细记录《Brotato》的经济系统、商店机制、价格体系等。

---

## 一、经济系统概述

### 1.1 货币类型

游戏中只有一种货币：**材料 (Materials)**

- 显示为绿色数字
- 用于购买武器、道具
- 用于刷新商店
- 用于武器升级

### 1.2 经济循环

```
击杀敌人 → 掉落材料 → 波次结束 → 商店消费 → 进入下一波
```

```
基础掉落2 → 实际掉落 = 2 × 1.5 = 3
```

## 三、商店系统

### 3.1 商店界面

商店在每波结束后出现，包含：
- **4个物品栏位**（可购买武器或道具）
- **升级选项**（消耗经验值选择属性）
- **刷新按钮**（重新随机物品）
- **锁定功能**（保留物品到下一波）

### 3.2 商店刷新

| 操作 | 费用 | 说明 |
|------|------|------|
| 首次刷新 | 3材料 | 基础费用 |
| 第2次刷新 | 5材料 | +2 |
| 第3次刷新 | 7材料 | +2 |
| 第N次刷新 | 3+(N-1)×2 | 递增 |

**每波重置刷新费用**

### 3.3 物品锁定

- 点击物品上的锁图标
- 锁定的物品保留到下一波
- 锁定免费，但占用栏位
- 购买后锁定自动解除

### 3.4 物品出售

- 已购买的武器/道具可以出售
- 出售价格 = 购买价格 × 50%
- 在商店界面点击已拥有物品出售

## 四、物品品质与价格

### 4.1 品质等级

| 等级 | 名称 | 颜色 | 属性倍率 | 价格倍率 |
|------|------|------|----------|----------|
| Tier 1 | 普通 | 白色 | ×1.0 | ×1.0 |
| Tier 2 | 优秀 | 绿色 | ×1.25 | ×1.5 |
| Tier 3 | 稀有 | 蓝色 | ×1.5 | ×2.0 |
| Tier 4 | 史诗 | 紫色 | ×1.75 | ×2.5 |

### 4.2 品质出现概率

基础概率：

| 品质 | 基础概率 |
|------|----------|
| 普通 | 55% |
| 优秀 | 30% |
| 稀有 | 12% |
| 史诗 | 3% |

### 4.3 幸运属性影响

```
调整后概率 = 基础概率 × (1 + 幸运调整)

幸运对每个品质的调整：
- 普通：-幸运×0.3%
- 优秀：+幸运×0.15%
- 稀有：+幸运×0.1%
- 史诗：+幸运×0.05%

例：幸运100%时
普通：55% × (1-0.3) = 38.5%
优秀：30% × (1+0.15) = 34.5%
稀有：12% × (1+0.1) = 13.2%
史诗：3% × (1+0.05) = 3.15%
```

### 4.4 波次影响品质

随着波次推进，高品质物品出现率提升：

| 波次 | 史诗加成 | 稀有加成 |
|------|----------|----------|
| 1-5 | +0% | +0% |
| 6-10 | +1% | +2% |
| 11-15 | +2% | +4% |
| 16-20 | +3% | +6% |

## 七、经济策略

### 7.1 收支平衡分析

**典型一波收入（难度1，波次10）：**
```
击杀敌人：60×1.5 = 90
波次奖励：15
精英击杀：3×5 = 15
总计：约120材料
```

**典型支出：**
```
购买2个道具：约60材料
刷新1次：5材料
总计：约65材料
结余：约55材料
```

### 7.2 阶段策略

| 阶段 | 建议支出比例 | 说明 |
|------|--------------|------|
| 前期(1-5) | 武器60%，道具40% | 快速获得武器 |
| 中期(6-12) | 武器30%，道具50%，升级20% | 开始升级武器 |
| 后期(13-20) | 武器10%，道具40%，升级50% | 专注升级 |

### 7.3 刷新策略

| 情况 | 建议 |
|------|------|
| 商店无合适物品 | 刷新1-2次 |
| 材料充足 | 可多刷新寻找高品质 |
| 材料紧张 | 保守消费，减少刷新 |
| 寻找特定道具 | 锁定有用物品，逐步刷新 |

### 7.4 收获流玩法

以农夫角色为例：

| 波次 | 收获属性 | 材料加成 |
|------|----------|----------|
| 1 | 25% | +25% |
| 5 | 40% | +40% |
| 10 | 70% | +70% |
| 15 | 115% | +115% |
| 20 | 175% | +175% |

后期材料收入可达普通玩家的2-3倍。

## 八、特殊经济机制

| 道具 | 效果 |
|------|------|
| 银行 | 每波结束获得当前材料5%的利息 |
| 投资 | 消耗50材料，下一波开始获得75 |

### 8.2 商人角色

- 商店物品-15%价格
- 开局+25材料
- 适合快速成型Build

### 8.3 探险家角色

- 每波额外2个箱子
- 箱子内容+50%
- 减少商店依赖
