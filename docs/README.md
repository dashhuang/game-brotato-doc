## 开发优先级

### P0 - 核心系统 (必须)
- [ ] 战舰移动控制
- [ ] 自动射击系统
- [ ] 基础敌舰AI
- [ ] 战役阶段生成
- [ ] 舰桥升级系统
- [ ] 基础补给站

### P1 - 重要系统 (核心体验)
- [ ] 多武器装备
- [ ] 武器升级合成
- [ ] 模块效果系统
- [ ] 多战舰系统
- [ ] 难度系统
- [ ] 舰长立绘显示

### P2 - 扩展内容 (完整体验)
- [ ] 舰长好感度系统
- [ ] 舰长语音系统
- [ ] 剧情系统
- [ ] 成就系统
- [ ] 无尽深空模式

### P3 - 锦上添花 (优化)
- [ ] 舰长皮肤系统
- [ ] 特效完善
- [ ] 音效完善
- [ ] 排行榜
- [ ] 教程系统
