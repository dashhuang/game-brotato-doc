## 1. 游戏概述

### 1.1 基本信息

| 属性 | 内容 |
|------|------|
| **游戏名称** | 星河舰长 (Galaxy Captain) |
| **游戏类型** | 俯视角自动射击 + Roguelike + 舰长养成 |
| **题材** | 二次元太空战舰 |
| **平台** | PC、iOS、Android |
| **美术风格** | 帅气科幻风（明日方舟/机战风格） |

### 1.2 游戏背景

公元3025年，人类的星际联邦遭遇了前所未有的危机——虫族入侵、AI叛乱、海盗猖獗、异星帝国觊觎。在这个动荡的时代，你将作为新任舰队指挥官，率领各型战舰和英勇的女性舰长们，在星河中书写传奇。

### 1.3 核心特色

- **自动射击**：战舰自动锁定攻击最近敌人，专注于战术走位
- **多武器装备**：每艘战舰最多同时装备**6种舰载武器**
- **丰富的Build构建**：48艘战舰 × 67种武器 × 135种模块
- **快节奏战役**：每场战斗20-90秒，一局游戏约20-30分钟
- **Roguelike元素**：随机模块、战舰成长、永久解锁
- **舰长养成**：48位女性舰长的立绘、语音、好感度、剧情

### 1.4 世界观概述

| 势力 | 描述 |
|------|------|
| **星际联邦** | 人类主导的星际政府，玩家所属阵营 |
| **虫族舰队** | 宇宙生物群落，拥有有机战舰 |
| **机械叛军** | 失控的AI舰队，曾是联邦的自动化部队 |
| **星际海盗** | 亡命之徒组成的掠夺者联盟 |
| **异星帝国** | 神秘的外星文明，拥有超越人类的科技 |

## 2. 核心玩法循环

### 2.1 单局游戏流程

```
选择战舰 → 选择初始武器 → 进入战役阶段1
     ↓
[战斗阶段] 在星域中生存，击毁敌舰获取战术经验和能量晶体
     ↓
[舰桥升级] 消耗战术经验选择属性提升
     ↓
[补给站] 使用能量晶体购买武器/模块/升级
     ↓
进入下一战役阶段 → 重复直到第20阶段或战舰损毁
     ↓
结算 → 舰长好感度增加 → 解锁新内容 → 返回母港
```

### 2.2 战斗阶段详解

- **持续时间**：第1阶段20秒，逐渐增加至第20阶段90秒
- **敌人刷新**：敌舰从屏幕边缘持续跃迁进入
- **Boss刷新**：特定阶段会出现敌方旗舰/母舰
- **掉落物**：
  - 战术经验（蓝色能量球）
  - 能量晶体（绿色结晶）
  - 修复包（生命恢复）
  - 补给箱（额外奖励）

### 2.3 舰桥升级详解

每次升级可从**4个随机属性**中选择1个进行强化：

| 属性类型 | 提升内容 |
|----------|----------|
| 舰体容量 | +3 最大护盾 |
| 护盾再生 | +1 每5秒回复 |
| 能量吸收 | +1% 攻击回盾概率 |
| 火力输出 | +5% 伤害加成 |
| 射击频率 | +5% 攻速 |
| 精确打击 | +3% 暴击率 |
| 装甲强化 | +1 装甲值 |
| 规避系统 | +3% 规避率 |
| 战场运势 | +5% 幸运值 |
| 资源回收 | +5% 晶体掉落 |
| 火力范围 | +10 攻击范围 |
| 引擎推力 | +3% 移动速度 |
| 近程强化 | +8% 近程炮加成 |
| 导弹强化 | +8% 导弹加成 |
| 能量武器 | +8% 激光加成 |
| 舰载指挥 | +8% 舰载机加成 |

- **独特的初始属性加成/减益**
- **独特的被动技能**
- **对应的女性舰长**（立绘、语音、剧情）
- **推荐的Build方向**
- **需要满足的解锁条件**

## 4. 属性系统

### 4.1 基础属性详解

#### 生存类属性

| 属性 | 说明 | 计算方式 | 上限 |
|------|------|----------|------|
| **最大护盾 (Max Shield)** | 战舰最大护盾值 | 基础10 + 加成 | 无上限 |
| **护盾再生 (Shield Regen)** | 每5秒回复护盾量 | 数值/秒 = 属性值/5 | 无上限 |
| **能量吸收 (Energy Drain)** | 攻击回盾概率 | 每次攻击 × 吸收% | 100% |
| **装甲值 (Armor)** | 减少受到的伤害 | 伤害减免 = 装甲值 | 无上限 |
| **规避率 (Evasion)** | 完全规避攻击概率 | 规避率 = 规避% | 60%/90% |

#### 攻击类属性

| 属性 | 说明 | 计算方式 | 上限 |
|------|------|----------|------|
| **火力输出 (Firepower)** | 基础伤害加成 | 武器伤害 × (1 + 加成%) | 无上限 |
| **射击频率 (Fire Rate)** | 武器攻击频率 | 基础冷却 / (1 + 攻速%) | 无上限 |
| **精确打击 (Critical)** | 暴击概率 | 暴击率% | 100% |
| **致命伤害** | 暴击伤害倍率 | 默认200%，可提升 | 无上限 |
| **火力范围 (Range)** | 攻击/拾取范围 | 基础范围 + 加成 | 无上限 |
| **穿透力 (Piercing)** | 弹道穿透敌舰数 | +1 = 多穿1艘敌舰 | 无上限 |

#### 特殊类属性

| 属性 | 说明 | 影响 |
|------|------|------|
| **近程强化** | 近程炮专属加成 | 仅对近程炮生效 |
| **导弹强化** | 导弹专属加成 | 仅对导弹类武器生效 |
| **能量武器** | 激光专属加成 | 激光/离子/等离子武器 |
| **舰载指挥** | 舰载机专属加成 | 提升舰载机/无人机威力 |
| **引擎推力** | 战舰移动速度 | 影响走位灵活性 |
| **战场运势** | 稀有掉落概率 | 影响商店/掉落品质 |
| **资源回收** | 能量晶体掉落加成 | 提升经济收入 |

### 4.2 属性计算公式

#### 伤害计算

```
最终伤害 = 武器基础伤害 
         × (1 + 火力输出%) 
         × (1 + 武器类型加成%)  // 近程/导弹/激光/舰载机
         × (1 + 致命伤害%)  // 如果暴击
         - 敌舰装甲
```

#### 攻击间隔计算

```
实际攻击间隔 = 武器基础间隔 / (1 + 射击频率%)
```

#### 装甲减伤

```
实际受到伤害 = 原始伤害 - 装甲值
(最低伤害 = 1)
```

## 14. 数值设计原则

### 14.1 边际效应

属性堆叠存在**边际递减**：
- 第一个+10%伤害 → 实际提升10%
- 叠加到+100%时，再+10% → 实际只提升5%

### 14.2 阈值效应

某些属性存在**关键阈值**：
- 规避率60%（普通上限）
- 规避率90%（隐身舰上限）
- 攻速达到某值可秒杀敌舰

### 14.3 平衡设计

```
DPS = (武器伤害 × 火力加成 × 类型加成) / 攻击间隔 × 暴击系数
生存时间 = 最大护盾 / (敌舰DPS × (1-规避率) - 装甲减免 - 回复)
```

## 15. 美术与音效风格

### 15.1 美术风格

- **整体风格**：帅气科幻、机械美学
- **色调**：冷色调为主（深蓝、银白、暗紫）
- **特效**：霓虹光效、激光轨迹、爆炸碎片
- **战舰设计**：流线型、机械感、辨识度高
- **舰长设计**：英气飒爽、军装风格

### 15.2 音效设计

| 类型 | 说明 |
|------|------|
| BGM | 电子/合成器风格，史诗感 |
| 战斗音效 | 激光、爆炸、导弹发射 |
| UI音效 | 科技感提示音 |
| 舰长语音 | 日语/中文配音（可选） |

### 15.3 屏幕反馈

- **受创**：屏幕边缘红光闪烁
- **护盾耗尽**：警报音效 + 红色警告
- **暴击**：加大伤害数字 + 特效
- **Boss出现**：屏幕震动 + 警告提示 + 舰长语音
