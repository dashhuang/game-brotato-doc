# 星河舰长 资源与补给系统详解

> 本文档详细记录《星河舰长》中的资源获取、补给站购买和经济机制。

---

## 一、资源类型

### 1.1 能量晶体 (Energy Crystal)

| 属性 | 描述 |
|------|------|
| **图标** | 💎 绿色六边形结晶 |
| **用途** | **双重功能**：积累升级 + 购买武器/模块 |
| **持久性** | 仅当局有效 |

> **设计说明**：能量晶体同时充当"经验值"和"货币"两种角色。收集足够晶体后触发舰桥升级，同时晶体也用于补给站购买。这意味着玩家需要在"升级属性"和"购买装备"之间做出权衡。

### 1.2 好感度点数 (Affection Points)

| 属性 | 描述 |
|------|------|
| **图标** | 💗 粉色爱心 |
| **用途** | 舰长养成 |
| **持久性** | 永久保存 |
| **获取时机** | 战斗结算后 |

## 二、能量晶体获取

### 2.1 击毁敌舰获取

| 敌舰类型 | 基础掉落 | 概率 |
|----------|----------|------|
| **普通敌舰** | 1-3 | 50% |
| **精英敌舰** | 5-10 | 80% |
| **Boss** | 30-100 | 100% |

### 2.2 补给箱获取

| 箱子类型 | 晶体数量 | 出现概率 |
|----------|----------|----------|
| **普通补给箱** | 5-15 | 常见 |
| **精英补给箱** | 15-30 | 少见 |
| **传奇补给箱** | 30-50 | 稀有 |

### 2.3 战役阶段奖励

| 阶段范围 | 通关奖励 |
|----------|----------|
| 1-5 | 5-10 晶体 |
| 6-10 | 10-15 晶体 |
| 11-15 | 15-20 晶体 |
| 16-20 | 20-30 晶体 |

### 2.4 资源回收加成

```
实际获取晶体 = 基础晶体 × (1 + 资源回收%)
```

**示例**：
- 资源回收+50%
- 击毁敌舰掉落2晶体
- 实际获取：2 × 1.5 = 3 晶体

## 三、舰桥升级系统

### 3.1 升级机制

能量晶体在用于购买的同时，也会**累计升级进度**。当收集的晶体达到升级阈值时，触发舰桥升级。

> **重要**：升级消耗的是"累计收集量"，不是"当前持有量"。购买物品不影响升级进度。

### 3.2 晶体-升级对照表

| 2 | 15 | 25 |
| 3 | 20 | 45 |
| 4 | 30 | 75 |
| 5 | 40 | 115 |
| 6 | 55 | 170 |
| 7 | 70 | 240 |
| 8 | 90 | 330 |
| 9 | 110 | 440 |
| 10 | 135 | 575 |
| 11+ | +25每级 | ... |

### 3.3 升级奖励

每次升级时，从**4个随机属性**中选择1个进行提升：

| 属性类型 | 提升内容 |

## 四、补给站系统

### 4.1 补给站界面

每个战役阶段结束后进入补给站，包含：
- **物品区域**：4个随机物品（武器或模块）
- **刷新按钮**：花费晶体刷新物品
- **锁定按钮**：锁定物品不被刷新
- **战舰信息**：当前属性面板
- **继续按钮**：进入下一阶段

### 4.2 物品刷新机制

#### 基础刷新规则
- 每次显示**4个物品**
- 物品可能是武器或模块
- 已拥有的武器显示**"升级"**标记

#### 刷新费用

| 刷新次数 | 费用 |
|----------|------|
| 第1次 | 3 晶体 |
| 第2次 | 5 晶体 |
| 第3次 | 7 晶体 |
| 第4次 | 9 晶体 |
| 第N次 | 3 + (N-1)×2 晶体 |

*注：每个阶段刷新次数重置*

### 4.3 物品稀有度概率

#### 基础概率（无运势加成）

| 稀有度 | Tier | 概率 |
|--------|------|------|
| 标准型 | 1 | 55% |
| 改良型 | 2 | 30% |
| 精英型 | 3 | 12% |
| 旗舰型 | 4 | 3% |

#### 战场运势影响

```
标准型概率 = 55% - (运势 × 0.3%)
改良型概率 = 30% + (运势 × 0.15%)
精英型概率 = 12% + (运势 × 0.1%)
旗舰型概率 = 3% + (运势 × 0.05%)
```

**示例**（运势+50）：
- 标准型：55% - 15% = 40%
- 改良型：30% + 7.5% = 37.5%
- 精英型：12% + 5% = 17%
- 旗舰型：3% + 2.5% = 5.5%

### 4.4 物品锁定

- 每个物品可单独锁定
- 锁定的物品在刷新时保留
- 锁定**不消耗**晶体
- 锁定的物品在下一阶段**不会保留**

### 4.5 武器升级

- 购买已拥有的相同武器会**升级**该武器
- 升级将武器提升1个Tier
- Tier 4为最高，无法再升级
- 升级价格 = 该Tier武器的正常价格

| 操作 | 说明 |

## 七、经济策略

### 7.1 早期阶段 (1-5)

| 策略 | 说明 |
|------|------|
| **优先** | 补充武器槽位 |
| **建议** | 买便宜的T1武器填满6槽 |
| **避免** | 过度刷新补给站 |

### 7.2 中期阶段 (6-15)

| 策略 | 说明 |
|------|------|
| **优先** | 升级核心武器 |
| **建议** | 锁定高Tier武器，下次购买升级 |
| **投资** | 资源回收模块提升收益 |

### 7.3 后期阶段 (16-20)

| 策略 | 说明 |
|------|------|
| **优先** | 完善Build |
| **建议** | 购买关键模块补齐弱点 |
| **允许** | 多次刷新寻找特定物品 |

### 7.4 经济向Build

| 核心模块 | 效果 |
|----------|------|
| 资源回收器×3 | +30 资源回收 |
| 高效回收 | +15 资源回收 |
| 经济学家 | +25%资源回收，-10%价格 |
| 粒子炮×6 | 每把击毁+1晶体 |

**预期收益提升**：200-300%

## 八、难度对经济的影响

### 8.1 难度与掉落

| 危险等级 | 掉落倍率 | 敌舰数量 |
|----------|----------|----------|
| 0 | ×0.8 | ×0.7 |
| 1 | ×1.0 | ×1.0 |
| 2 | ×1.1 | ×1.15 |
| 3 | ×1.2 | ×1.3 |
| 4 | ×1.3 | ×1.5 |
| 5 | ×1.4 | ×1.75 |

### 8.2 经济平衡

高难度下：
- 敌舰更多 → 更多掉落机会
- 但战斗更危险 → 需要更强装备
- 整体收益略高，但需要更快决策
