# Brotato 角色系统详解

> 本文档详细记录《Brotato》中所有角色的完整信息，包括属性、技能、解锁条件和推荐玩法。

---

## 一、角色系统概述

### 1.1 角色数量
- **基础角色**：1个（全能者，默认解锁）
- **可解锁角色**：47个
- **总计**：48个角色

### 1.2 角色属性结构

每个角色包含以下信息：
- **初始武器**：开局自带的武器
- **初始道具**：开局自带的道具（部分角色有）
- **属性加成**：正面效果
- **属性减益**：负面效果
- **特殊能力**：独特的被动技能
- **解锁条件**：如何解锁该角色
